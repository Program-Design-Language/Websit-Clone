ツイッターのお題「CSV編集」


ツイッターに出されたお題をやってみるシリーズです。
今回のお題は、CSV編集です。


ツイートへのリンク

https://twitter.com/i/status/1253307552802074625



VBSにしてみた

VBAでも良いのですが、たまにはVBSも良いかと思い作ってみました。



Option Explicit

Dim objFSO
Set objFSO = CreateObject("Scripting.FileSystemObject")

Dim var
For Each var in WScript.Arguments
　　Call WriteCsv(objFSO.getFile(var).Path)
Next

Set objFSO = Nothing

Sub WriteCsv(argFile)
　　'Set objFSO = CreateObject("Scripting.FileSystemObject")
　　Dim inFile, outFile
　　Dim rec, i, j, ary1, ary2, ary3

　　Set inFile = objFSO.OpenTextFile(argFile, 1, False)
　　Set outFile = objFSO.OpenTextFile(Replace(argFile, ".csv", "_new.csv"), 2, True)

　　Do Until inFile.AtEndOfStream
　　　　rec = inFile.ReadLine
　　　　ary1 = Split(rec, ",")
　　　　ary2 = Split(ary1(0), "／")
　　　　ReDim ary3(UBound(ary1))
　　　　For i = 0 To UBound(ary2)
　　　　　　ary3(0) = ary2(i)
　　　　　　For j = 1 To UBound(ary1)
　　　　　　　　ary3(j) = ary1(j)
　　　　　　Next
　　　　　　outFile.WriteLine Join(ary3, ",")
　　　　Next
　　Loop

　　inFile.Close
　　outFile.Close
End Sub



せっかくVBSなので、複数ファイルドラッグにも対応しています。
○○.csv
↓
○○_new.csv
として、編集後のcsvを出力しています。

特に解説はしませんが、VBSの場合、デバッグが大変ですね。
VBAである程度作成して、それを基にしてVBSに書き直す方が良いと思います。

ツイッターのお題に対する回答なので、とりあえずやってみたというものです。
したがって、コードの書き方で特段に工夫したところとかは特にありません。
