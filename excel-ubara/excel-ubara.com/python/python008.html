第8回.文字列操作（str型）


プログラミングでは文字列（テキスト）を扱う事は非常に多く最も基本的な処理と言えます。
Pythonでは文字列はstr型になります。
このstr型に対する操作として非常に多くの機能がPythonには用意されています。

文字列リテラル

'シングルクォートと"ダブルクォート

Pythonでは、文字列の囲みとして、'シングルクォートと"ダブルクォートのどちらでも使えます。
この「Python入門」では"ダブルクォートを使っています。
このサイトのメインコンテンツがVBAなので、"ダブルクォートにしておいた方が都合がよいというだけの理由です。

エスケープ文字

文字列中に、囲み文字と同じ文字が入っていると都合が悪いことになります。
"ダブルクォートを使った場合は、"ダブルクォート
'シングルクォートを使った場合は、'シングルクォート
これらの文字を文字列中に記述する場合は、エスケープ文字を使う必要があります。
その他、特殊文字にもエスケープ文字が必要になります。

Pythonでのエスケープ文字には、\（バックスラッシュ、日本語環境では¥）を使います。

エスケープ文字一覧
\' シングルクォート
\" ダブルクォート
\t タブ
\n 改行
\r キャリッジリターン
\\ バックスラッシュ

※この他にも用意されていますが、普通に使うと思われるものを抜粋しました。

s = "Pythonの\nエスケープ文字の\"\\\"について"
print(s)
会話モードでやってみると



\n：改行
\"："
\\：\

上記はWindows日本語環境です。

raw文字列

囲みのクォート文字の前に、rをつけるとraw文字列として、書かれた文字列のまま処理されます。

s = r"Pythonの\nエスケープ文字の\"について"
print(s)



三連クォート

Pythonでは、'シングルクォートまたは"ダブルクォートを3個つなげることで複数行の文字列を扱う事が出来ます。
三連クォートの間は、タブや改行やクォートが文字列として扱われます。
また、三連クォートの間はインデントも適用されません。

s = """Pythonの
"エスケープ文字"の'\\'について"""
print(s)


この三連クォートをコメントして使用する場合があります。

s="test"
""" 三連クォートをコメントとして使います。
これはコメントです。
"""
print(s)

第6回.for文とイテラブルオブジェクト
プログラムとは処理手順であり、突き詰めれば条件分岐しながら繰り返し処理を行うものです。繰り返し処理はプログラミング言語に無くてはならない処理になります。Pythonでの繰り返し処理のうち、今回はfor文の説明になります。

インデックスは先頭が0から始まります。

P y t h o n

-6 -5 -4 -3 -2 -1

文字列[index]

文字列のスライス

シーケンス（文字列、リスト、タプル等）から部分的に取得するものをスライスと呼びます。
スライスの指定は、
文字列[start:stop]
文字列[start:stop:step]

startからstop-1までが取得されます。
stopには文字数（最大インデックス+1）まで指定する場合が出てきます。
stepは飛び飛びで取得する場合に使用します。



stopの1つ手前までが範囲になる点に注意してください。
インデックス範囲外の数値を指定してもエラーにならず、有効な範囲内の文字が取得されます。


format()関数

組み込み関数としてformat()が用意されています。
引数の値を指定した書式文字列で変換して返します。

format(value[, format_spec])

valueには、元の値となる、str、int, floatを指定します。
format_specには、書式指定文字列を指定します。
書式指定の詳細については以下を参照してください。
公式ドキュメント - 書式指定ミニ言語仕様

print(format("右詰め","*>10")) #右詰め*埋め
print(format("中央寄せ","^10")) #中央寄せ空白埋め
print(format(12345,",")) #カンマ区切り
print(format(12.3456,".2f")) #小数2桁で丸め
print(format(0.1234,".2%")) #%で小数2桁



format()メソッド

文字列を引数の書式指定で変換します。
文字列中の{}が、引数に指定された値に置換されます。
以下の機能があります。
・渡した順に置換
・位置を指定して置換
・キーワード指定して置換
・:コロンに続けて書式指定

print("{}と{}と{}".format("a","b","c")) #渡した順に置換
print("{2}と{0}と{1}".format("a","b","c")) #位置指定に置換
print("{p}と{v}".format(v="VBA",p="Python")) #キーワード指定
print("{:,}".format(12345)) #書式指定



f-string（フォーマット済み文字列リテラル）

f-string（フォーマット済み文字列リテラル）はPython3.6から導入されたものです。
メソッドformat()より簡単に書けるようになっています。

文字列中に変数や式を{}でくくって指定します。
実行時に変数や式が評価され置換されます。
{}の中に、:（コロン）に続けて書式指定文字列を指定できます。

s1 = "Python"
s2 = "VBA"
print(f"右詰め{s1:*>10}") #右詰め*埋め
print(f"中央寄せ{s1+s2:*^10}") #中央寄せ空白埋め
d = 12345.678
print(f"{s1+s2}カンマ,{d:,}") #カンマ区切り
print(f"{s1:8}小数2桁,{d:.2f}") #小数2桁で丸め
d = 0.12345
print(f"{s1}%小数2桁,{d:.2%}") #%で小数2桁


Python3.8からは、変数名や式を出力してくれる機能が追加されました。
変数や式に続けて=を記述します。
メソッドも指定できます。

name = "Python"
print(f"{name=}")
print(f"{name.lower()=}")
print(f"{name.upper()=}")

isalphaisalpha()文字列中の全ての文字が英字で、かつ1文字以上あるならTrueを、そうでなければFalseを返します。
英字は、Unicode文字データベースで"Letter"として定義されているもので、すなわち、一般カテゴリプロパティ"Lm"、"Lt"、"Lu"、"Ll"、"Lo"のいずれかをもつものです。
なお、これはUnicode標準で定義されている"Alphabetic"プロパティとは異なるものです。

isdecimalisdecimal()文字列中の全ての文字が十進数字で、かつ1文字以上あるならTrueを、そうでなければFalseを返します。
十進数字とは十進数を書くのに使われる文字のことで、たとえばU+0660(ARABIC-INDICDIGITZERO)なども含みます。
正式には、Unicodeの一般カテゴリ"Nd"に含まれる文字を指します。

たとえばU+2155(VULGARFRACTIONONEFIFTH)。

upperupper()全ての大小文字の区別のある文字が大文字に変換された、文字列のコピーを返します。
なおs.upper().isupper()は、sが大小文字の区別のある文字を含まなかったり、結果の文字のUnicodeカテゴリが"Lu"ではなく例えば"Lt"(Letter,titlecase)などであったら、Falseになりえます。
zfillzfill(width)長さがwidthになるようASCII'0'で左詰めした文字列のコピーを返します。
先頭が符号接頭辞('+'/'-')だった場合、'0'は符号の前ではなく後に挿入されます。
widthがlen(s)以下の場合元の文字列を返します。
