ドキュメントのダウンロード

広告

PEARに関するドキュメントもダウンロードしておきます。下記のURLから開始します。

http://pear.php.net/

上部メニューの中にある「Documentation」と書かれたリンクをクリックして下さい。

開いたページを下の方へスクロールして下さい。

「Japanese」と書かれたブロックがあります。その中の「Many HTML files zip」と書かれたリンクをクリックして下さい。

ダウンロードが開始されます。任意の場所に保存して下さい。

ダウンロードしたファイルは圧縮ファイルとなっています。今回は「d;¥php¥」ディレクトリに「peardoc」ディレクトリを作成し、その中に解凍したファイルを配置しました。解凍すると「html」ディレクトリが作成され、ファイルはそのディレクトリ内に含まれています。

「html」ディレクトリ内にある「index.html」ファイルをブラウザで開いて下さい。

PEARに関するドキュメントを参照することができます。
