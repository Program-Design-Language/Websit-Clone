Strawberry Perlのダウンロード及びインストール

広告

Strawberry Perl のダウンロード及び Windows10 環境へのインストール方法について解説します。 2021 年 1 月現在の最新バージョンは Strawberry Perl 5.32.0.1 です。

目次

  1. Strawberry Perlをダウンロードする
  2. Strawberry Perlをインストールする
  3. インストールとPATHの設定が正常に完了したか確認する

Strawberry Perlをダウンロードする

最初に Strawberry Perl のダウンロードを行います。次のURLから開始します。

・https://strawberryperl.com/

今回は最新版の 64bit 環境向けの MSI installer 付きのものをダウンロードします。画面右下に表示されている strawberry-perl-5.32.0.1-64bit.msi をクリックしてください。

Strawberry Perl のダウンロードが開始されます。任意の場所に保存しておいてください。

なお他のバージョンや ZIP 形式のものをダウンロードしたい場合は ZIP, Portable, special editions をクリックしてください。

バージョンやファイル形式毎にダウンロードできるページが表示されます。

Strawberry Perlをインストールする

続いてインストールを行います。ダウンロードしたファイルをダブルクリックして下さい。インストールが開始されます。

ライセンスの確認です。よく読んで頂いて同意できる場合には「I accept the terms in the License Agreement」をチェックして下さい。チェックが終わりましたら Next をクリックして下さい。

インストールするディレクトリを設定します。今回はデフォルトで表示されたディレクトリから変更してあります。設定が終わりましたら Next をクリックしてください。

最終確認です。インストールして宜しければ Install をクリックして下さい。

次のような画面が表示されればインストールは完了です。 Finish をクリックして下さい。

インストールとPATHの設定が正常に完了したか確認する

Strawberry Perl をインストールすると自動的に必要な PATH が設定されるようです。

インストールが正常に完了したかどうか確認します。コマンドプロンプトを起動して下さい。(コマンドプロンプトの起動方法については「コマンドプロンプトを起動する」を参照されて下さい)。

次のように実行して下さい。

perl -v

Perl のバージョン情報が表示されれば Strawberry Perl のインストール及び PATH の設定は正常に完了しています。
