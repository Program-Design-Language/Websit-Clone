jQuery 1.5.1 のダウンロード

広告

それではjQueryのダウンロードを開始します。次のURLから開始します。

  • http://jquery.com/

画面上部に表示されている「Download」メニューをクリックして下さい。

開いたページを少し下へスクロールして頂いて「Download jQuery」と書かれたところをご覧下さい。最新バージョンから過去のバージョンまでダウンロード可能となっています。今回は「Current Release」のところにある jQuery 1.5.1 をダウンロードします。

「Minified」と「Uncompressed」の2種類が用意されています。どちらも内容は同じなのですが「Minified」の方は改行などが削除されておりサイズが小さくなっていますが、ファイルの中身を確認するには不向きです。「Uncompressed」の方はサイズが大きいですが読みやすくなっています。内容を確認するには「Uncompressed」、実際に読み込むには「Minified」を使用するといいかと思います。(次のページで実際に違いを確認します)。

今回は両方ダウンロードしておきます。それぞれのリンク先を右クリックして「名前を付けて保存」などでダウンロードしておいて下さい。

「Uncompressed」の方は「jquery-1.5.1.js」、「Minified」の方は「jquery-1.5.1.min.js」というファイル名になります。

ダウンロードは以上で終了です。
