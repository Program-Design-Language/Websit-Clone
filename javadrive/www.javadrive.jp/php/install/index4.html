PHPのドキュメントを参照する

広告

PHP の公式サイトで用意されている PHP に関するドキュメントを参照する方法を解説します。 PHP のドキュメントはオンライン上で閲覧することもできますし、ダウンロードしておいて参照することもできます。

目次

  1. オンラインでドキュメントを参照する
  2. ドキュメントをダウンロードする

オンラインでドキュメントを参照する

最初にオンラインでドキュメントを参照する方法です。次の URL へアクセスして下さい。

・https://www.php.net/

画面上部に表示されている「Documentation」をクリックして下さい。

「Documentation」に関するページが表示されます。

オンライン上にあるドキュメントを閲覧したい場合には「View Online」の右側にある「Japanese」と書かれたリンクをクリックして下さい。

日本語で記述された PHP に関するドキュメントを閲覧することができます。

ドキュメントをダウンロードする

ドキュメントはダウンロードしておくこともできます。ダウンロードしておけば、インターネットにつながっていない状況でも参照することができます。先ほどオンラインでドキュメントを参照した時と同じ画面を表示して下さい。

「Downloads」の右側にある「documentation downloads」と書かれたリンクをクリックして下さい。

ドキュメントのダウンロード用ページが表示されます。

ドキュメントの形式として「Single HTML file」「Many HTML files」「HTML Help file」「HTML Help file (with user notes)」の 4 種類があります。今回は複数の HTML ファイルに分かれている「Many HTML files」を選択しました。「Japanese」と書かれている列の「tar.gz」をクリックします。

ダウンロードが開始されますので任意の場所に保存して下さい。ダウンロードはこれで終了です。

ダウンロードしたファイルを展開すると「php-chunked-xhtml」というディレクトリが作成され、その中にドキュメントファイルが格納されます。今回は PHP をインストールしたディレクトリに展開しました。

ディレクトリの中に「index.html」というファイルが含まれていますのでブラウザで開いて下さい。

オンラインで閲覧したものと同じ PHP に関するドキュメントをダウンロードして閲覧することができます。
