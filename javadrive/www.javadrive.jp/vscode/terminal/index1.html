ターミナルを開く/ターミナルを閉じる

広告

Visual Studio Code ではウィンドウ内でターミナルを開くことができます。ここでは統合ターミナルの基本操作として、ターミナルの開き方や閉じ方。複数のターミナルを利用する方法などについて解説します。

目次

  1. ターミナルを開く
  2. 複数のターミナルを開く
  3. ターミナルを分割する
  4. ターミナルを閉じる/ターミナルを強制終了する

ターミナルを開く

ターミナルを開くには Visual Studio Code の「ターミナル」メニューの中にある「新しいターミナル」をリックしてください。

ウィンドウの下部にターミナルが表示されます。 Windows 環境の場合、デフォルトで起動するターミナルは Windows PowerShell です。

なお Visual Studio Code でフォルダを開いている場合、ターミナルを開いた時のカレントディレクトリは開いているフォルダとなります。

複数のターミナルを開く

ターミナルは一度に複数開くことができます。ターミナルの画面右上に表示されている「新しいターミナル」アイコンをクリックしてください。

新しいターミナルが同じ位置に開きました。それぞれのターミナルは独立して使用します。

ターミナルを切り替える場合は、ターミナルの画面右上に表示されているドロップダウンメニューをクリックしてください。現在開いているターミナルの一覧が表示されますので、表示したターミナルを選択してください。

ターミナルを分割する

ターミナルが表示されている画面を分割して別のターミナルを開くこともできます。ターミナルの画面右上に表示されている「ターミナルの分割」アイコンをクリックしてください。

ターミナルが表示されていた場所が左右に分割されて、それぞれターミナルが表示されました。それぞれのターミナルは独立して使用します。

同じ手順でさらに分割して利用することもできます。

ターミナルを閉じる/ターミナルを強制終了する

ターミナルを閉じるには、開いているターミナルで用意されているコマンドを使って閉じます。例えば Windows PowerShell であれば exit コマンドでターミナルを閉じることができます。

現在表示されていたターミナルが閉じました。(複数のターミナルを開いていた場合は表示されていたターミナルだけが閉じます)。

ターミナルを分割している場合に、いずれかのターミナルを閉じたい場合もこの方法で閉じることができます。

ターミナルを強制終了することもできます。ターミナルの画面右上に表示されている「ターミナルの強制終了」アイコンをクリックしてください。

現在表示されていたターミナルが閉じました。(複数のターミナルを開いていた場合は表示されていたターミナルだけが閉じます)。

ターミナルを分割している場合に「ターミナルの強制終了」アイコンをクリックすると、表示されているターミナルの中で、アクティブになっているターミナルだけが強制終了されます。
