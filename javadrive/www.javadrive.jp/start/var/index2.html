変数を宣言する

広告

変数を利用するには最初に変数の宣言を行う必要があります。変数の宣言は、変数の名前と変数に格納されるデータのデータ型を指定して行います。ここでは Java のプログラミングで変数を宣言する方法について解説します。

目次

  1. 変数の宣言
  2. 同じデータ型の値を格納する複数の変数をまとめて宣言
  3. 変数を宣言せずに変数に値を代入する

変数の宣言

変数を宣言する場合の書式は次のようになっています。

データ型 変数名;

Java では複数の変数を利用することができますので、それぞれの変数を区別するために変数名を付ける必要があります。そして変数に格納される値のデータ型を指定します。 Java では変数を宣言する時に指定したデータ型とは別のデータ型の値を変数に格納することはできません(例外あり)。

例えば int 型の値を格納する height という名前の変数と、 char 型の値を格納する c という名前の変数をそれぞれ宣言するには次のように記述します。 

int height;
char c;

これで変数の宣言が完了しました。

※ 変数名には命名規則があります。詳しくは「変数名のつけ方(識別子)」を参照してください。

※ Java 10 から var を使用することで変数の宣言時にデータ型を省略できるようになりました。詳しくは「varを使って変数の宣言時にデータ型を省略する(型推論)」を参照してください。

同じデータ型の値を格納する複数の変数をまとめて宣言

同じデータ型の値を格納する複数の変数を宣言する場合、まとめて宣言することができます。

データ型 変数名1, 変数名2, ...;

例えば int 型の値を格納する height と old という 2 つの変数を宣言する場合は次のように記述していました。 

int height;
int old;

まとめて変数の宣言をする場合、変数名をカンマ(,)で区切って続けて記述します。先ほどの例は次のように記述することができます。 

int height, old;

どちらの書き方がいいというわけではありません。まとめて記述することでプログラムが少しだけコンパクトになりますが、別々に変数を宣言した場合と比べて少し分かりにくくいと感じる場合もあります。コメントを記述する場合などは別々に宣言された方がコメントを書きやすくなります。

サンプルコード

それでは簡単なサンプルプログラムを作って試してみます。テキストエディタで次のように記述したあと、 JSample2-1.java という名前で保存します。 

class JSample2_1{
  public static void main(String[] args){
    int old, height;
    old = 20;
    height = 184;

    System.out.println("年齢：" + old);
    System.out.println("身長：" + height);
  }
}

コンパイルを行います。

javac -encoding UTF-8 JSample2_1.java

その後で、次のように実行してください。

java JSample2_1

変数 old と height を宣言し、それぞれの変数に値を代入したあと、変数に格納されている値を取り出して画面に出力しました。

変数を宣言せずに変数に値を代入する

変数は値を格納したり格納した値を参照したりして利用します。値を格納するには、変数を宣言し、その後で変数に値を代入するのですが、変数を宣言せずに変数に値を代入するとエラーとなります。

次の例をみてください。 

class JSample2_2{
  public static void main(String[] args){
    count = 0;
  }
}

変数を宣言せずに変数に値を代入しています。このプログラムをコンパイルしてみると「エラー: シンボルを見つけられません」というエラーが表示されます。

変数に値を代入する前には必ず宣言を行ってください。
