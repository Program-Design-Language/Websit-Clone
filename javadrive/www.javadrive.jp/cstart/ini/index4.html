プログラムの整形

広告

プログラムはテキストエディタに記述していくことは既に書きましたが、記述方法についてもう少し細かく見ておきます。また今までの使っていたサンプルを見て下さい。 

#include <stdio.h>

int main(void){
  printf("Hello World!¥n");
  return 0;
}

プログラムを記述する場合には、プログラムを読みやすくするために改行や空白を入れることが出来ます。例えば上記のサンプルは次のように記述することもできます。 

#include <stdio.h>

int main(void){
printf("Hello World!¥n");
return 0;
}

今までの例ではプログラムの行の前に空白をいくつか入れていました。これは単に見やすくするためにしていたものです。main関数の中に記述されたプログラムがどこからどこまでなのか行の先頭に空白を入れることで分かりやすくなるためです。

また次のように記述することも出来ます。 

#include <stdio.h>

int main(void){printf("Hello World!¥n");return 0;}

今までの例では1つの処理毎に改行を行ってきましたが、改行は必要なものではありません。どこからどこまでが1つの処理なのかを区別するために最後にセミコロン(;)を付ける規則になっています。その為、改行をあえて行わなくても処理の単位は区別することができます。

ただ、プログラムというものは正確に記述すればいいというだけではありません。後からメンテナンスを行う時のことを考えて見やすい記述も心がけるべきです。最初に書いた人ではない他の人が後でプログラムを読むこともあります。 

#include <stdio.h>

int main(void){
  printf("Hello World!¥n");
  return 0;
}

特に決まったルールはありませんが、プログラムは1つの処理毎に改行し、インデント(行の字下げをおこなうこと)を使ってどこからどこまでの行が1つのまとまりとなっているのかを表すようにします。
