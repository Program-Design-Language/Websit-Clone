とほほの厳選ツール集(WWWブラウザ)

トップ > とほほの厳選ツール集 > WWWブラウザ

Internet Explorer

★★★ [Win][Mac] 無料
http://www.microsoft.com/Japan/

Microsoft社のブラウザ。Windows に標準で添付されています。現在世界で最も標準的に使用されているブラウザです。（2002.10.13更新）

Netscape Navigator

★★ [Win][Mac][UNIX] 無料
http://home.jp.netscape.com/

Netscape社のブラウザ。Mosaicを開発したアンドリーセン氏が設立した会社で、すこし前までは世界ナンバーワンのシェアを誇っていましたが、現在は Internet Explorer に完全に抜かれてしまいました。Netscape Navigator 1.0/2.0/3.0、Netscape Communicator 4.*、Netscape 6.0/7.0 など、製品名もいろいろ変更されています。（2002.10.13更新）

Opera

★★ [Win][Mac][UNIX] 無料版/製品版
http://www.jp.opera.com/

ノルウェーの Opera Software 社が開発・販売する第三のブラウザ。IE や Netscape にくらべて軽快に動作し、マウスアクション機能など、いろいろ便利な機能がサポートされている。無料版は常に広告が表示されるが、4,800円（アカデミックは 2,800円）を支払うことにより、広告を表示しないようにすることができる。（2002.10.13更新）

Mosaic

★ [Win][Mac][UNIX] 無料
http://www.ncsa.uiuc.edu/SDG/Software/Mosaic/

WWWの黎明期にNCSA(当時)のマーク・アンドリーセン氏が開発したブラウザ。Windows, Macintosh, UNIX版がフリーソフトで提供されている。当時は「ホームページを見る」ことを「Mosaicを見る」と呼ぶほど流行っていた。（というより、これしか無かった）

Mozilla

★ [Win][Mac][UNIX] 無料
http://www.mozilla.org/

元 Netscape の開発メンバを中心とした組織 mozilla.org が開発・提供するブラウザ。Netscape 6.0/7.0 も Mozilla.org が開発したエンジン（Gecko）をベースとしているが、Mozilla は、Gecko をベースとした、Netscape とは別のブラウザ。（2002.10.13更新）
