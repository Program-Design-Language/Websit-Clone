ページ全体の色(背景色)を変えるには

ページ全体の色を変えるには、<body> に background-color を指定します。

HTML
<style>
body {
    background-color: #ffc;
}
</style>

文字の色を変えるには

文字の色を変えるには、color を用います。<span> や <div> など、任意のタグに指定することができます。

HTML
<span style="color:red">赤い文字</span>

文字の色を一度に変えるには

<head>～</head> の間に次のようなスタイルシートを記述します。body は本文全体、a:link は <a> によるリンク、a:visited は訪問済みリンク、a:active はアクティブ中(主にはクリック中)のリンクを意味します。

HTML
<html>
<head>
<title>TEST</title>
<style>
body { color: #333333; }
a:link { color: #009900; }
a:visited { color: #006600; }
a:active { color: #ff0000; }
</style>
</head>
<body>
  :
</body>
</html>

もしくは、外部スタイルシートファイルを用いて、複数のページのスタイルを一度に変更することもできます。「スタイルシートの指定方法」を参照してください。

マウスを乗せたときのリンクテキストの色を変えるには

リンクテキストにマウスを乗せたときの色を変えるには、スタイルシートの hover を用います。

HTML
<style>
a:hover { color: #ff0000; }
</style>
