リスト(箇条書き)を書くには

<ul> は順序無しリスト(Unorderd List)を意味します。

HTML
<ul>
<li>カレーライス
<li>スパゲティ
</ul>
表示
  • カレーライス
  • スパゲティ

<ol> は順序リスト(Orderd List)を意味します。

HTML
<ol>
<li>カレーライス
<li>スパゲティ
</ol>
表示
  1. カレーライス
  2. スパゲティ

リスト先頭のマークや数値を変えるには

リストの先頭の丸や数字の部分を変更するには、スタイルシートの list-style-type を指定します。

HTML
<ol style="list-style-type:lower-alpha">
<li>おでん
<li>うどん
</ol>
表示
  1. おでん
  2. うどん
HTML
<ul style="list-style-type:circle">
<li>おでん
<li>うどん
</ul>
表示
  • おでん
  • うどん

定義リストを書くには

用語のその説明の組みからなる定義リストを書くには、<dl>、<dt>、<dd> を用います。</dt>、</dd> の終了タグは省略可能です。

HTML
<dl>
<dt>HTML</dt>
<dd>HTMLとは・・・</dd>
<dt>CSS</dt>
<dd>CSSとは・・・</dd>
</dl>
表示
HTML
HTMLとは・・・
CSS
CSSとは・・・
