BGMを鳴らすには

音を鳴らすには以前は <bgsound> や <embed> や <object> を利用していましたが、今では <audio> が利用されています。Chrome, Firefox, Opera, Safari, Edge および、Internet Explorer 9 以降で利用可能です。

HTML
<audio src="./audio/sound.mp3" autoplay loop></audio>

曲を再生・停止したりするには

曲を再生したり一時停止したりするには、pause() や play() を用います。

HTML
<audio id="audio1" src="./audio/sound.mp3" autoplay loop></audio>
<input type="button" value="Pause" onclick="document.getElementById('audio1').pause()">
<input type="button" value="Play" onclick="document.getElementById('audio1').play()">
