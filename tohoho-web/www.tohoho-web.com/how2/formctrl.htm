サブミットボタンを押したことにするには

JavaScript でサブミットボタンを押したことにするには、Form オブジェクトの submit() メソッドを用います。

JavaScfipt
function func() {
  document.form1.submit();
}

テキスト入力欄のテキストを得るには

JavaScript で、<input> の type="text"、type="file"、type="hidden" や、<textarea> の入力値を得るには、value プロパティを参照します。

JavaScfipt
<script>
function func() {
  alert(document.form1.t1.value);
  alert(document.form1.f1.value);
  alert(document.form1.h1.value);
  alert(document.form1.a1.value);
}
</script>
<form name="form1" onsubmit="func()">
  <input type="text" name="t1" value="TEXT1">
  <input type="file" name="f1">
  <input type="hidden" name="h1" value="HIDDEN1">
  <textarea name="a1">TEXTAREA</textarea>
  <input type="submit" value="OK">
</form>
