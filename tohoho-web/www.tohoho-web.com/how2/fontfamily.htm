Windowsのフォント

Windows 標準のフォントとして以下のものなどがあります。等幅フォントはすべての文字が同じ幅ですが、プロポーショナルフォントは文字によって幅が異なります。

ＭＳ 明朝Microsoft 等幅明朝。
