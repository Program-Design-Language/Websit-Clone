コアモジュール

Drupal 6.4 を標準でインストールした際には、下記のコアモジュールが含まれています。

必須モジュール

モジュール 説明
System システムを管理します。
User ユーザアカウントを管理します。
Block ブロックを管理します。
Node ノードを管理します。
Filter コンテンツを表示する際に、様々なフィルタ処理を行います。

基本モジュール

モジュール 説明
Locale 英語以外の言語環境を利用可能にします。
Content transration コンテンツの翻訳をサポートします。
Update status [管理セクション]→[リポート]→[入手可能な最新版] で、Drupal本体、モジュール、テーマのアップデート状況を確認できるようになります。定期的に実行したり、最新版情報をメールで通報することもできます。
Menu [管理セクション]→[サイトの構築]→[メニュー] で、ナビゲーションメニュー、プライマリリンクメニュー、セカンダリリンクメニューなどのメニューの表示・非表示を設定することが可能になります。
Help [管理セクション]→[ヘルプ] で、各モジュールのヘルプを参照できるようになります。
Trigger [管理セクション]→[サイトの構築]→[トリガ] で、「新しいコンテンツが登録された時」などのイベントに対して、「管理者にメールを送信する」などのアクションを指定可能にします。

コンテンツタイプモジュール

標準でサポートされる「ページ」、「ストーリー」に加えて、下記のコンテンツタイプを追加することが可能です。

モジュール 説明
Blog 「ブログエントリ」コンテンツタイプの投稿を可能にします。
Forum 「フォーラムトピック」コンテンツタイプの投稿を可能にします。
Book 「Book page」コンテンツタイプの投稿を可能にします。
Poll 「投票」コンテンツタイプの投稿を可能にします。

コンテンツに関連するモジュール

モジュール 説明
Comment コンテンツに「コメント」をつけられるようにします。
Taxonomy 「タクソノミー」をサポートします。
Aggregator 他のサイトから、RSS/RDF/Atom を用いてコンテンツ情報を収集します。
Ping コンテンツが更新されたことを、他のサイトに通知します。
Tracker 未稿。
PHP filter 未稿。
Throttle 未稿。
Blog API 未稿。

管理系モジュール

モジュール 説明
Database logging システムイベントをデータベースに記録します。
Syslog システムイベントを syslog に記録します。
Statistics サイトへのアクセス情報を記録します。

ユーザ管理系モジュール

モジュール 説明
OpenID OpenID によるログインを可能にします。
Profile ユーザープロフィールのカスタマイズを可能にします。
Contact 個人用、サイト用のコンタクトフォームを利用可能にします。

その他の便利なモジュール

モジュール 説明
Color 色設定ツールを利用可能にします。
Upload コンテンツ作成・編集時に添付ファイルのアップロードを可能にします。コンテンツの編集画面で [ファイルの添付] 機能が追加されます。[管理セクション]→[サイトの環境設定]→[ファイルアップロード] で最大ファイルサイズや、アップロード可能なファイルの拡張子などの指定が可能です。アップロードされたファイルは、http://～/sites/default/files/(ファイル名) に保存されます。
Search 「検索」ブロックを利用可能にします。検索を有効にした際には、cron.php で検索インデックスを生成する必要があります。[管理セクション]→[サイトの環境設定]→[検索の設定] や、[管理セクション]→[リポート]→[検索フレーズの上位] が利用可能になります。
Path URLの別名（エイリアス）を利用可能にします。
