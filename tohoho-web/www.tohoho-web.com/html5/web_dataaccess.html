ローカルデータアクセス

HTML5 では、Cookie よりもより柔軟で、より多くのアプリケーションデータをクライアント側(ローカル)に保存・参照できる仕組みがサポートされています。

  • Web Storage
  • Web SQL Database
  • Indexed Database API

それぞれの特徴は下記の様になります。
