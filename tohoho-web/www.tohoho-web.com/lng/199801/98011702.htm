とりあえず
#!/usr/local/bin/perl
print "Content-type: text/plain\n\n";
foreach ( sort keys ( %ENV ) ){
　　print "$_ = $ENV{$_}\n";
}
でスクリプト側に渡っている全ての環境変数が見られます。
