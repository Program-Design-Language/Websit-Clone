メール本文中のリンクからはちょっと難しいかもしれませんが、
リンククリック時に別ウィンドウを開くパターンであれば、
JavaScriptでウィンドウの大きさを指定することができます。

<A HREF="#" onClick="open('xx.html','xx','width=200,height=200'); return false">
XXXX</A>
