Omi 1998/04/27(月) 05:43:08
実験的には、以下で画像が変わります。
<A HREF="#"
   onMouseOver="img1.src = 'xxx2.gif'"
   onMouseOut= "img1.src = 'xxx3.gif'">
<IMG NAME="img1" SRC="xxx1.gif" BORDER=0></A>
iE4 と NN4 で確認しました。
NN4 では、最初の画像エリアに合わせて、後の画像の縦横比が変わります。
iE4 では、画像の大きさが異なり、相対的にマウスが Overしたり Outしたりすると、
マウスを少し動かす毎に、チラチラするので、WIDTH HEIGHT の項が必要です。

参考になれば幸いです。
