hiro-kim 1998/04/20(月) 10:08:20
Remote Access Service。モデムが接続されたファイルサーバがあり，そこにリモートアクセスするものです。プロトコルとしてFTPを使わなくても，Windowsのファイル共有がそのまま使えてわかりやすいというのが利点です。不特定多数向けのサービスではなく，会社組織でのデータ共有方法ですね。
