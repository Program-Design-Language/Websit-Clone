ふじ 1999/03/17(水) 16:25:39
Aのウィンドウに名前が付いているなら（他のウィンドウから開いたりして）、
ウィンドウの名前.document.フォームの名前.テキストボックスの名前.value = 求めた数値;

Aのウインドウに名前が付いていないなら、NN3,IE4以降対応になってしまいますが
window.opener.document.フォームの名前.テキストボックスの名前.value = 求めた数値;

でよろしいかと。
