INPUT TYPE="button" の押下時にCGIの関数を呼ぶにはどうしたらいいのでしょうか？教えて下さい。（onClickでJavaScriptの記述はできるみたいですが）
ポケ鬼！！ 1999/03/12(金) 17:37:05
TYPE="BUTTON"にこだわるのなら、

<FORM ACTION="hoge.cgi" METHOD="POST" NAME="hoge">
<INPUT INPUT="BUTTON" VALUE="送信" onClick="submit(hoge);">
</FORM>

普通は、

<FORM ACTION="hoge.cgi" METHOD="POST">
<INPUT INPUT="SUBMIT" VALUE="送信">
</FORM>

です。
