B-Cus 1999/03/03(水) 01:34:35
> $mailhead .= "From: $FORM{'EMAIL'}\n";
を
　$mailhead .= "From: $FORM{'EMAIL'} ($FORM{'NAME'})\n";
か
　$mailhead .= "From: "$FORM{'NAME'} <$FORM{'EMAIL'}>\n";
とするとどうでしょうか。どちらを選ぶかは好きにしてください。
RFC的にはどちらでもいいです(多分)。

あと、生JISをヘッダに含めたくなければ、$FORM{'NAME'} を
MIMEエンコードしてください。つまり、
　=ISO-2022-JP?B?.......=?=
となるようにしてください。ま、結局自分だけに送られてくる
わけですから、あなたが気にしないなら生JISでもいいでしょう。
