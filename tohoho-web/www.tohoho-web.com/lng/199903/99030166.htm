telnet でログインできますか？
もしできるなら、telnet でログインした後、カレントディレクトリ
を、データファイルを設置したいディレクトリに移して（cd コマン
ドを使います）、
$ touch wwwboard.dat
でwwwboard.datという名前の空ファイルを作成できます。

cd コマンドは MS-DOS と同じ要領です。
