open ホスト名 … 指定のホストに接続
ls (ファイル・ディレクトリ名)… ファイル一覧表示
cd (ディレクトリ名) … リモートのカレントディレクトリを変更
lcd (ディレクトリ名) … ローカルのカレントディレクトリを変更
pwd … リモートのカレントディレクトリを表示
get … ファイルをローカルに転送
put … ファイルをリモートに転送
binary … バイナリモードに移行
ascii … アスキーモードに移行
mget  … ワイルドカードを使ってファイルを指定できるget
mput  … ワイルドカードを使ってファイルを指定できるput
hash … 転送マーク表示/非表示(トグル)
prompt … mget/mput使用時に、確認を求めるかどうか設定(トグル)
!(コマンド) … ローカルでコマンドを実行
　　例えばローカルのカレントディレクトリを表示するなら
　　!pwd (UNIXの場合)とか !cd (MS-DOSの場合) など。
