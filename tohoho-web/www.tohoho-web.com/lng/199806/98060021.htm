1つのホームページに、複数の変数と、それぞれに対応するSUBMITボタンを作成し、SUBMITボタンで送信する変数を指定する方法をご教授願います。
目的は、次の2点です。
1.複数の項目で、任意の順番にデータを入力可能
2.入力されたデータに対応するデータをファイルからロードし、それを参照しながら次のデータを入力可能

<INPUT TYPE=SUBMIT NAME="n1" VALUE="乙">
<INPUT TYPE=SUBMIT NAME="n2" VALUE="甲">
のようにすれば、
・「乙」のボタンを押したら n1 が "乙" になり n2 が未定義
・「甲」のボタンを押したら n2 が "甲" になり n1 が未定義
となるので、どのボタンを押したかが区別できます。
ようするに、押していないボタンの値は未定義になります。
