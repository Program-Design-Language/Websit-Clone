<FORM onSubmit="location.href='ＵＲＬ'; return false;">
と書く代わりに、
<FORM onSubmit="location.hash='#どこそこ'; return false;">
でOKだと思います。

<INPUT TYPE=submit ....>の代わりに<INPUT TYPE=button ...>でもいいかもしれません。
(別に変わらないといえば変わらないんですが)
buttonの時はonSubmitを書かずに、
<INPUT TYPE=button .... onClick="location.hash='#どこそこ'">
と書きます。

↓location.hashについて
http://www.tohoho-web.com/js/location.htm

>やはり NAME をつかうのかなあと思って

location.hashは<A HREF="#...">を使う時と同じですのでNAMEも入れてください。
