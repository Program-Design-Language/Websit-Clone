asp内でVbScriptを使用して、ある文字列を区切文字によって、分割し
てとりだそうとしたところ、上手く機能しません。
例）
Dim w_item(4)
Dim w_msg                '内容は、aaaa;bbbb;cccc;dddd;eeee

w_item=Split(w_msg,";")  '変数w_msg内の文字列を区切文字";"で
                          w_itemに分割格納しようとする
＜現象＞
w_itemに何も格納されていない

わかる方教えてください！
ＧＥＮ 1998/12/14(月) 15:12:06

Dim w_item(4) → Dim w_item

とすれば、大丈夫だと思いますよ。
ちなみに、私が試した状態を下記にコピーしますので、試してみて下さい。

Dim w_item
Dim w_msg
Dim i

w_msg = "aaaa;bbbb;cccc;dddd;eeee"
w_item=Split(w_msg,";")

For i = 0 To UBound(w_item)
  MsgBox w_item(i)
Next
