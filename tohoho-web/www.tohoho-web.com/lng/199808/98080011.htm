Masa [E-Mail] 1998/08/09(日) 22:28:05
<SCRIPT language="JavaScript" src="xxxx.js">
という風に、外部のJavaScriptファイルを読み込む事が
出来ます。だから、xxxx.jsのなかに、

function WriteLink(){
document.write("<A HREF=\"index.htm\">トップへ<\/A>");
}

とか、書いておけば、あとは<BODY>の中で
<SCRIPT>
WriteLink();
</SCRIPT>
と実行するだけでいいでしょう。ただし、どのブラウザが
対応しているかちょっと解りません。＞どなたか
いちおう、NN4とIE4では確認しました。とほほさんによれば
NN3でもオッケーとか。
