JavaScriptのソースを書いていると１行がすごく長〜くなってしまう時があります。
でもソース中で改行すると場所によってはエラーになります。
ソース中で改行する方法はありますか？
mura [E-Mail] 1998/08/22(土) 20:34:15
文字列なら分割できますよ。

==
document.writeln('この文章、長くなるので分割が必要です。');
==

↓

==
document.writeln(
'この文章、長くなるので' +
'分割が必要です。');
==

長くなるオブジェクトを変数にしてしまうという方法もあります。
==
parent.frame1.document.formX.email.value = '';
parent.frame1.document.formX.url.value = 'http://';
==

↓

==
var fm;
fm=parent.frame1.document.formX;
fm.email.value = '';
fm.url.value = 'http://';
==

あと、インデントしている場合はインデントをスペース1つに抑えるとか。

もちろん長いままにしておくという選択肢もアリです。
でも一長一短かな??長すぎると文法ミスが発見しにくくなりますし。
