cgi-bin/ の / は、cgi-bin がディレクトリであることを示すために
付けたのですが、確かに分かりづらかったですね。(^_^;)
「cgi-bin」という名前のディレクトリを作成してください。

また、.cgi のファイルを開くには、Windows であれば、メモ帳を起
動して、[ファイル]→[開く]でそのファイルを指定してください。
