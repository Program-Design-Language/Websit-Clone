とりあえず具体例を書いてみました。
質問の意味を取り違えていたら指摘して下さい。

-------- フレームを呼び出すHTML --------
<HTML>
<FRAMESET COLS="50%, 50%">
<FRAME NAME="LFRAME" SRC="lframe.html">
<FRAME NAME="RFRAME" SRC="rframe.html">
</FRAMESET>
</HTML>

------ 左フレームのHTML(lframe.html) ------
<HTML>
<BODY>
<A HREF="JavaScript:window.parent.RFRAME.document.FRM01.submit();">
<IMG SRC="イメージファイル名"></A>
</BODY>
</HTML>

------ 右フレームのHTML(rframe.html) ------
<HTML>
<BODY>
<FORM NAME="FRM01" ACTION="参照先のURL" METHOD=POST>
<INPUT TYPE=text NAME="TXT01">
</FORM>
</BODY>
</HTML>
