とほほ 1998/03/28(土) 22:09:03
escape()は、% や ? などの、URLとして特別な意味を持つ記号を、
%XX (XXは16進数のコード) に変換してくれる関数です。本当は
CGI側でも、%XX を % に戻してやる処理が必要なのですが、これは、
近いうちにホームページ上で公開しますね。
