＞A HREF="#"　←これはステータスバーに表示される文字
＞onClick="window.open('xx.html'　←これは小ウィンドウ自体のHTML
＞です・・・よね。あってますか？

その通りにございます。　＃ HREF="#" は勉強になりました。＞とほほさん

＞そこで、
＞'XX'　←これは何を指すのですか？

この 'XX' ってのは、新たに開くウィンドウにつけられる名前です。とほほさんがおっしゃった、
＞'XX'の部分を同じにすれば同じ別ウィンドウに、
＞別の名前にすればそれぞれ別のウィンドウに表示されます。
というのは、'XX' をすべてのリンクで同じにすれば、違うリンクをクリックしても、その 'XX' というウィンドウに次々とページ（いまの場合、画像）が上書きされる形で表示される、ということです。（かな？）　最初の質問の後者（「ひとつの小ウィンドウが開いていて」）はこっち。
逆に、すべてのリンクに対して別々なウィンドウを開きたい場合は、各リンクに対して 'XX1' とか 'XX2' とかのように指定してやればよいです。同じく前者（「それぞれ別の小ウィンドウ」）。

＞<SCRIPT language="JavaScript"><!-- //--></SCRIPT>
＞というのは省略していい種類のｓｃｒｉｐｔなんですか？

オッケーっす。

で、補足なんですが、「それぞれ別の小ウィンドウ」の場合、次のようにするといいかも。
## ここは <HEAD> タグ内に ##
<SCRIPT LANGUAGE="JavaScript">
<!--
function fileopen(file) { window.open(file+'.html',file); }
//-->
</SCRIPT>

## 各リンクの部分では ##
<A HREF="#" onClick="fileopen('index3')"><IMG SRC="xxx.gif" BORDER="0"></A> // "index3" が「小ウィンドウ」に表示するページ

さて、気になるブラウザ依存については、ここで使われているのはたいていのもので正常に動くと思います。
