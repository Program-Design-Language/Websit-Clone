一柳ででん 1998/03/25(水) 15:53:20
　世の中には心無い人が多いですね。ネット荒らしには私も散々悩まされています。
　で、犯人特定なのですが、実際には様々なケースがあります。いずれにしても、その発言があった時刻と送信元ＩＰアドレス（ドメイン名）は必須でしょう。それを元にいくつかのパターンを紹介します。

（１）一般プロバイダの生ＩＰの場合
　一番簡単です。そのプロバイダに連絡すれば犯人が使ったアカウントまでは割り出せるからです。ただし、それだけでは犯人を特定するには不十分です。盗まれたアカウントだったというケースも多いですし、銀行振込による決済を行っているプロバイダの場合、架空名義であることもしばしばです。
　当然、海外のプロバだったら英語による問い合わせが必要です。

（２）プロバイダの公開プロクシの場合
　公開プロクシの場合も管理者に時刻を伝えれば犯人のアカウントは割り出せます。でも、管理者不明のプロクシも多く、海外のプロクシも無数に存在するため、何重にもクシ刺しにされている場合などはかなり困難な作業となります。幸運にもアカウントが割れたとしても、（１）と同じケースが考えられます。

（３）企業、学校のプロクシ
　これも比較的簡単です。管理がずさんでなければ外に対するアクセス・ログくらいは取っているでしょうから、管理者に問い合わせれば一発で犯人が割れます。相手が閉じた空間なだけに個人特定まで成功することがしばしばあります。ただし、大企業や国立大学、官公庁などは管理がいい加減なところが多いです。

（４）カフェ・Ｑ２プロバイダ
　システム上、犯人の特定は不可能に近いです。そのＩＰアドレスを握っていた人物の情報がどこにも記録されていないのですから。この場合は泣き寝入りするか、全て締め出す以外に方法はありません。
