Syn [E-Mail] 1999/04/12(月) 14:59:31
DynamicHTML の Dynamic は「躍動的に HTML を表示する」…
んじゃなくて 「動的に」という意味です :-)
Dynamic の反対語は Static (静的) です。
ふつうにウェブページを読み込んで表示すると 書かれている内容は
変わることはありません (Static) が DynamicHTML だと ボタンを
押したりすることによりページのコンテンツを動的に変化させること
ができます。
(折りたたまれた文章が表示されたり LAYER で表示されたり)
もちろんそれを制御するのは JavaScript や VBScript なんですが
そういう手法で見栄えを操作すること全般を DynamicHTML という
ようです。
