の様にメールを出す文は
<a href="mailto:なんとか.jp?Subject=TITLE">めーる</a>
とするとサブジェクトに”TITLE”が自動的に入ってくれる
というのを最近知ったのですが、どうも挙動が変なんです。
というのも、メールアドレスやサブジェクトの文字列の組み合わせや、
長さによって、うまくいったり、途中でとぎれたり、
ブラウザ（IE4）がハングアップしたりと、一筋縄ではいきません。
何か法則性や、決まり事でもあるのでしょうか？
