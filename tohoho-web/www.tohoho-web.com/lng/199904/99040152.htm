TOM2 1999/04/21(水) 11:57:58
<a href="リンク先のアドレス" target="_blank">りんく</a>
で、別のウィンドウが開きます。

またリンク集などで、一度新しいウィンドウを開いて、リンク先をずっと
その別のウィンドウに表示させる場合は、" target="_blank"の「_blank」
の箇所を好きなウィンドウ名を入れられると良いと思います。
例えば、、、
<a href="リンク先のアドレス" target="new">りんく</a> <br>
<a href="リンク先のアドレス2" target="new">りんく2</a>

もし、私の解答がみかりんさんの質問の主旨からずれていたらスミマセン。
のあのあ 1999/04/21(水) 12:05:01
（１） <A HREF="xxx.html" TARGET="_blank">ほにゃらら</A>
　　　これは”思いっきり”嫌われます。

（２）<A HREF="xxx.html" TARGET="HOGE">ほにゃらら</A>
　　　<A HREF="yyy.html" TARGET="HOGE">もにゃらら</A>
　　　<A HREF="zzz.html" TARGET="HOGE">ふにゃらら</A>

　　　このように、"_blank"以外の名前を付ければ、複数のリンクが全部
　　　同じ新しいブラウザの中で切り替わります。でも、嫌われます。

一番いいのは、新しいブラウザで見る方法（右クリックでうんたらかんたら・・・とか、シフトキーでなんちゃらほにゃらら）を書いてあげることです。

＃ユーザは、自分の意志で新しいブラウザを開くことは出来ますが、
＃新しいブラウザで開く設定を解除することは出来ないのです。
＃重いときにさらにもう一枚ブラウザが上がって、マシンがハングし、
＃その人にとって重要なデータが消えてしまうかもしれません。
＃自分勝手な仕様で閲覧者に思わぬ不幸を招くことがあるということ
＃を忘れないでください。
無責任官庁 1999/04/21(水) 12:57:57
確かに自分の望まない事を勝手にされてしまうと、いやな感じする場合もありますが、
逆にそれを望んでる場合は、操作が面倒に感じたりするものです。
操作方法を書くにしても、ブラウザのメーカー・バージョンによって
千差万別だと思いますので、あまり現実的ではないでしょう。

ですから、選択の余地を残すということで、新しいブラウザを開く
リンクを付ける場合、開かない場合のリンクも残す、という方法がベストでしょう。
リンクをアイコンなどにして並べて表示したりすれば、
見た目もキレイでいいでしょう。
