いただいてましたが、どーしてもわからないところが出てきて
しまいました。教えてください。

クリッカブルマップのリンク先で、別のウインドウを表示させたくて
HEAD部に
function OpenHo(){
var win_ho=window.open
('office/honsya.html','honsya','toolbar=no,location=no,
status=no,menubar=yes,scrollbars=no,width=650,height=480')
win_ho.document.close();
}
と記述し、
クリッカブルマップ内で
<AREA SHAPE="rect" COORDS="2,71,126,170" onClick="OpenHo()">
と記述してみました。
エクスプローラではうまく表示されてのですが、
ネットスケープではなにも起こりません。
どこがまずいのでしょうか？
それともネットスケープでは使えない機能？？？

よろしくお願いします。
たこすけ 1999/04/23(金) 13:49:43
恐らく原因は<AREA>にHREFが無いから

onClickを取っちゃって「HREF="JavaScript:OpenHo()"」とするか
「HREF="何でも良い" onClick="OpenHo(); return false"」
としてみてはどうでしょう？
まさや [E-Mail] 1999/04/26(月) 11:05:46
[[解決]]
たこすけ様＞

回答をありがとうございます。
「HREF="JavaScript:OpenHo()"」のほうでうまくいきました。
「HREF="何でも良い" onClick="OpenHo(); return false"」だと
ウインドウが開かず、そのままリンク先に変わってしまうようです。
本当にありがとうございました。
