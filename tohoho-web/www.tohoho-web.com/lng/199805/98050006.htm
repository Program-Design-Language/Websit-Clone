そのときの状態がどうであれクリックさせてしまうわけだから、厳密に「ONにしたい」となると、ちょい工夫が必要。
フォームの名前が"f1",チェックボックスの名前が"c1"として、

if ( ! document.f1.c1.checked )  document.f1.c1.click();

のようにすれば、「ONにする」ということが可能かと思われます。
