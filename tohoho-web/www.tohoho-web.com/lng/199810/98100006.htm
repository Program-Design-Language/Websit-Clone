メイン画面があり、その中にリンクするページを書き込んでおいて、開く際に、別画面でどんどん開いていきたいのですが、そう言った方法はありますでしょうか？
みんこ 1998/10/02(金) 11:29:16
<A HREF="***.html" target="_blank">別ウィンドウでリンクします</A>
これで別ウィンドウが開いて、リンクします。

ただ、「どんどん」新しいウィンドウをひらいていくと、
環境によってはかなりきついかも。

<A HREF="***.html" target="newwin">とかにして、別ウィンドウに
名前（この場合はnewwin）をつけて指定すると、
新しく開いた方のウィンドウにリンク先が表示されます。
（target="newwin"をつけなければ元ウィンドウに表示されます）

既に『target="_blank"』で開いたウィンドウがあってもそれは変わりません
ちなみに一度に二つも三つものウィンドウを開くならばJavaScriptを使います
