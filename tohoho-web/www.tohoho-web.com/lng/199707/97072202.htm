とくめい 1997/07/25(金) 01:15:19
&func()はperlのスクリプトの中で呼び出します。
  #!/usr/local/bin/perl
  sub funcA { print "AAAAAA\n"; }
  sub funcB { print "BBBBBB\n"; }
  print "Content-type: text/html\n";
  print "\n";
  print "[[[\n";
  &funcB();
  &funcA();
  print "]]]\n";
なんていうスクリプト(a.cgi)を ACTION="a.cgi" で呼び出せば、
雰囲気が解るかもしれません。
