やっぱし・・・・
あとは、<IMG SRC="xxx.cgi">と、CGIを呼び出して、その中で

#!/usr/local/bin/perl
print "Content-type: image/gif\n";
print "Expires: ≪日付≫\n";
print "\n";
system("/bin/cat ≪GIFファイル≫");

ってな感じで、Expireキャッシュ期限付きのGIFイメージを返す
とか・・・・・・ちょっと面倒くさいかな。
