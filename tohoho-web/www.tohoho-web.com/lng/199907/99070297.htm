すべてのフォントを一括変更するなら，
http://www.tohoho-web.com/html/basefont.htm
です。

個々のフォントをまとめて設定したいなら，スタイルシートのクラス( http://www.tohoho-web.com/css/basic.htm#Class )を使って，

<style type="text/css">
<!--
.font1 { font-family: '……'; }
.font2 { font-family: '……'; }
-->
</style>

<p class="font1">これはfont1です。</p>
<p class="font2">これはfont2です。</p>

とすれば良いでしょう。
