sendmailを使ったことがないので、そのコマンドが全然
把握できていません。

blatでは、初回はコマンドラインから、
>blat -install xx.xxx.ne.jp username
でuser1としてログオンでき、次回からはusenameという
ユーザーとして、認識されます。

filename.txt というファイルをxxxxx@xxxx.xx.jpという相手に"テスト"というサブジェクトとして送るのは、
>blat filename.txt -s "テスト" -t "xxxxx@xxxx.xx.jp"
で簡単に送れます。

これをperlの中に書き込めばいいのでしょうけれど、perl
もまだまだよく分かっていないものですから。
