厳密には、HTML4.0 Transitionalであれば許される、でしょうか。
HTML文書のてっぺんに
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">
を付けておきませう。
ただし、size=3だからとて、個々のWWWブラウザで異なるサイズの
フォントを設定されていれば、違った大きさになってしまいますから、
あまり意味がないかもしれません。

><H1>-</H1>で囲まれた文字の大きさは何ptでしょうか？
それも、スタイルシートで指定されてはいかがかと。
<H1 style="font-size: XXxx">
とか何とか(XXxxは任意の値・単位を指定)。
そうされるつもりで、そもそもどのくらいの大きさになっているのかと
お尋ねなのだとしたら、結局は個々のWWWブラウザの設定次第でしょうね。

ちなみにMozilla4.6では設定されたフォントサイズの2倍のボールドに
なるようです。してみると、HTMLかBODYにfont-sizeを指定しておいて、
H1には2emを指定する手が使えるかもしれませんね(未確認)。
