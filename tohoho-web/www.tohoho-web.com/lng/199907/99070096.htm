予めお断り。私は Windows版の Apache はほとんど使ったことがないです。

Windows95(98)でApacheを動かしているのなら、Windowsを終了すれば一緒に
終わってくれませんか？

NTだと、サービス（UNIXでいうところのデーモン) として動作させていると
ログオフしてもApacheは起動したままになります。それを止めたければ、
「コントロールパネル」→「サービス」で、Apache サービスを停止して、
スタートメニューから Uninstall Apache Service(NT only) を選べば
サービスから外れます。
