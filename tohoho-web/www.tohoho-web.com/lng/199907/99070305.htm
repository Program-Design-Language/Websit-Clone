　こんな風にするといかがでしょうか。
　NN4では、ブロックの中に解読しきれないものがあると、
ブロック全体を無視してしまう傾向があります。
（それを逆手に取る方法のもアリだと思います(笑)）

　NNでは、borderの設定が、よく無視されてしまうのですが、
次のようにすると、うまく出てきました。

　NN4とIE4/5での表示の違いを見比べると、面白いかもしれません。
（Win版で確認しています）

　<LINK REL=STYLESHEET HREF="〜〜〜"> から *.css を読んでも、
きちんと表示されました。


<HTML><HEAD><STYLE TYPE="text/css"><!--
/* 共通 */
H1 {
background:#4242A3;
color:white;
width:610px;
font:normal normal normal 24pt/24pt sans-serif;
margin:20pt 0 8pt 0;
padding:0 0 0 0;
border-width:2pt 4pt 4pt 6pt;
border-color:#4242A3;
text-align:center;
}

/* NNでは無視されちゃいます。 */
H1 {
border-style:outset;
font:normal normal normal 24pt/24pt "ＭＳ Ｐゴシック",osaka,sans-serif;
}

/* IE4/5の独自拡張 */
H1 {
filter:Alpha(opacity=100,finishOpacity=80,style=1,startX=0,startY=0,finishX=0);
}
--></STYLE></HEAD><BODY>
<H1>スタイルシート 表示テスト</H1>
<BODY></HTML>
