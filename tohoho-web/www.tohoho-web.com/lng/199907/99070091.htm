<TABLE>
<TR>
<TD WIDTH="50%">（装飾）</TD>
<TD WIDTH=500 NOWRAP>（メイン）</TD>
<TD WIDTH="50%">（装飾）</TD>
</TR>
</TABLE>

ウィンドウが十分でないと（装飾）は半分切れたり、中途半端に表示されます。
（メイン）は最大限、守られます。

MSIE 5.0 は完璧に再現します。
Mozilla 4.6 は、条件によってはスクロールバーの現れることがあります。
（装飾）の中に大きな IMG があったり、別の WIDTH がある場合などです。
他のブラウザは未確認。まあ、レイアウトが崩れるとみて間違いないです。

MSIE 5.0 からの閲覧を前提にすれば、非常に有効な技でしょう。
