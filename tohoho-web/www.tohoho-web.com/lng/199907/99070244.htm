改訂版。上のだと前月の末日を出すのと，引数として月は0-11の値を要求する（JavaScriptの

<SCRIPT language="javascript">
function getLastDate(Y,M){
d1=new Date(Y, M-1, 1);
d1.setTime(d1.getTime()+1000*60*60*24*32);
d2=new Date(d1.getYear(),d1.getMonth(),1);
d2.setTime(d2.getTime()-1000);
return d2;
}
</SCRIPT>
