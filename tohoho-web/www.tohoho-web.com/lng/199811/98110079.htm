左のリストを選択されると、右のリストの対応された項目を
選択させるスクリプトをつくりました。

function Sect(frm){
frm.lst2.selectedIndex = frm.lst1.selectedIndex;
}

IE３なら選択された項目が表示されるようにスクロールするのですが、
NC３だと選択はされるのですが、表示されるのはトップからになります。
NC３でもリストの表示位置をかえる方法はないでしょうか。
やっぱりあきらめるしかないのでしょうか。
