1.Accessのクエリーで抽出したテーブルをaspファイルで表示する。
2.作成されたaspファイル(一覧表)から、各詳細ページへリンクして。
3.各詳細ページ同士「次の詳細ページへ」、「前の詳細ページへ」、
  ボタンを作りたい。
※詳細ページは1つのフォーマットで、表示する内容、画像、
  を変えるだけです。RequestQueryなどを使って表示。
  <A HREF="ファイル名.asp?画像=***&データ=***">って感じで
  1つのフォーマットファイルにデータを変える感じです。
  だから、「次へ」と｢前へ」の内容を変えるだけなんです。
