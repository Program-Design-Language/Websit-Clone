UNIXなら、制限をかけたいディレクトリに
.htaccessという名前のファイルを置く事で制限がかけれたと思います。

例えば、.comドメインからのアクセスだけを拒否するなら、

<Limit Get Post>
order deny,allow
allow from all
deny from .com
</Limit>

という内容の.htaccessファイルを作って置いときます。

以上の内容はプレンティスホール出版の「CGI入門」からです。

このファイルと.htpassedファイルを組み合わせると、
アクセスするのに、パスワードで制限をかける事ができたりもします。
