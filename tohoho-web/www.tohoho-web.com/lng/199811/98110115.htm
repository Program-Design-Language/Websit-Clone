Tiot 1998/11/27(金) 14:20:32
@INCに入っているディレクトリにあるライブラリならどこでも使えます。
print @INC;で表示できます。
このなかに、目当てのライブラリがあればそれぞれのディレクトリに
設置する必要はありません。
無いときは、スクリプトの最初に（requireやuseの前）
push (@INC, "library_path");
で、ライブラリを探索するディレクトリを追加できます。
unshiftを使ってもいいです（付け加えたディレクトリを優先的に探すようになる）。
