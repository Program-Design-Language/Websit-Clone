パーミッションでは期待の動作は行なえません。
public_html/.htaccess に

<Files ~ \.htaccess>
order deny,allow
deny from all
allow from 見せてもいいホスト(全面禁止ならこの行毎削除)
</Files>

でいかがでしょう。

おまけでこんなのは、
<Files ~ "\.(dat|log)$">
order deny,allow
deny from all
allow from hogehoge
</Files>

拡張子が'.dat'または'.log'を隠します。
ちなみにこれはApacheでの指定です。
