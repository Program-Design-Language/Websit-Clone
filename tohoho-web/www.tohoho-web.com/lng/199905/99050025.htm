例えば、下のようなソースを書き、
------------------------------------
<html>
<!--#exec cmd="/cgilib/hoge.pl 1"-->

なんたらかんたら

<!--#exec cmd="/cgilib/hoge.pl 2"-->

なんたらかんたら

<!--#exec cmd="/cgilib/hoge.pl 3"-->
</html>
------------------------------------

値に応じた画像やテキストを表示したいのです。

　このとき、SSIは上の行の実行が終了してから下の行の実行が始
まると考えても良いのでしょうか？
