配列の要素数を得るのにはどうしたらいいのでしょうか？
なんか現在、、

$i = 0;
foreach $a (@b){
    $i++;
}

というような間抜けなことをしています。どうやればいいか教えてください。
また、
foreach $a (@b[5..]){
     ・・・;
}

とやれば、配列の５番目の要素から最後の要素まで処理できるのでしょうか？、、、もし間違っていれば、配列の n 番目の要素から最後の要素まで処理を行う方法を教えてください。
ふじ 1999/05/15(土) 21:54:53
$num = @a;
で、@aの要素数が $num に入ります。
