[[解決]]
ページ構築後（文書：完了等とブラウザに表示された状態）
にdocument.write()を呼び出すと予期せぬエラーが起こる
と何かにありました
どうしてもページに書き込みたい場合はフォームを使うか
window.open()&document.open()
を使ってそこの画面に書き込むしかないようです
こんな感じで
<SCRIPT Language="JavaScript">
<!--
newWindow = window.open("","");
newWindow.document.open();
newWindow.document.write("<html><body>");
newWindow.document.write("書き込む内容");
newWindow.document.write("</body></html>");
newWindow.document.close();
//
-->
</SCRIPT>
ただし書き込む内容は変数や配列等を使って
一度に書き込んだ方がいいと思います。
