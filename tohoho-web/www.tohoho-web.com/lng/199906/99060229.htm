B-Cus 1999/06/23(水) 21:57:36
1. 管理者にcgiwrapを入れてくれと頼む。
2. 管理者にsuExecを入れてくれと頼む。
3. suidperl(が入ってるなら)を使う。スクリプトをchmod 4755。
4. 全部Cで書いてchmod 4755。
5. Cでwrapperを書いてchmod 4755。その中でperlを実行。

1か2が楽だし、まっとうな方法です。
