moci [E-Mail] 1999/06/08(火) 10:44:57
<A href="***.html"><IMG src="anime.gif" width="横幅" height="高さ" alt="***"></A>

のようにwidthとheightで画像サイズを指定しておけば、
画像が読み込まれる前からレイアウトが決定できます。
また、altで飛び先の情報が書かれていれば、画像を
読み込まない環境でも、どういうページへ飛ぶのか
知ることができます。

ただ、あきさんの書かれている「一部分」が気になります。
画像そのものがリンクでなく、場所によってリンク先を
変えるのであれば、MAPとAREAの方でしょうか↓

http://www.tohoho-web.com/html/map.htm
