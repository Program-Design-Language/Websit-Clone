配列の一番右の要素をいったんはずしておいて、
文字を追加書き込みしてから、また一番右端にさっきの要素を追加したいです。
open(IN,"log.dat");
@lines = <IN>;
close(IN);

@lines = reverse(@lines);
($lines,@lines) = @lines;
@lines = ($lines,aaaaaa,@lines);
@lines = reverse(@lines);

open(IN,">log.dat");
print IN "@lines";
close(IN);

いまはこうやってリバースを二回使ってごまかしているのですが、
ログが多くなってくると恐ろしく負荷がかかりそうで心配です。
なにかスマートなやり方があったら教えてください。
お願いします。
