Ａ１．バージョンが同じならかまいません。
Ａ２．実行したい順に書くのがいいでしょう。また、関数を呼び出したときにその関数がないとエラーが出るので、汎用的に使うものほど最初の方に書いた方がいいです。
例えば

function hoge(){
hoge2();
}
function hoge2(){
alert('こんばんは〜');
}

のような順番で書くのはやめた方がいいです。

バージョンが違うときは、古い順でしょうね。古い関数を上書きするという技をどこかでやってました。
