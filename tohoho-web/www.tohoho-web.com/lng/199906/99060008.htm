つまり〜、一枚のファイルで指定するから無理なので
はじめに左右で分けて、その中でさらに上下に分けないと
理想のフレームは多分切れません。

一枚目のファイルで真ん中からたてにぶったぎる。
その際境界線オン、リサイズオンにする。
左右のファイルで真ん中からよこにそれぞれぶったぎる。
その際両方境界線オフ、リサイズオフ。

これでできあがり。
