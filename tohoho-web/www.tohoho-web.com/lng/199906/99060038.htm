半角スペースが出てしまうのはきっと記述ミスでしょう。
多分その本のどこかで、

　　$value =~ s/\r*$|\n//g;

これに似た感じの処理があると思います。

その //g: のところの // の間に半角スペースを置いているのではないでしょうか？そうであるならば、そこはつめてください。

それで、15文字ずつで区切らなくても、自動改行できるようにすることは可能ですが…。
