画像やテキストをダイナミックに書き換える

トップ > How To > 画像やテキストをダイナミックに書き換える

説明

jQueryを用いて、Webページ上の画像やテキストなどのオブジェクトを書き換えることができます。

実例

下記の入力欄を変更して [OK] ボタンを押してみましょう。

This is a pen.

ソースコード

<script src="./lib/jquery-1.10.2.min.js"></script>
<textarea id="S1" cols=40 rows=5>This is a pen.</textarea>
<input type="button" value="OK" onclick="$('#D1').html($('#S1').val())">
<div id="D1" style="width:340px; height:80px; border:1px solid gray;">This is a pen.</div>
