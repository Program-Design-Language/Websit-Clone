スタイル

◆ element.style

要素のスタイルに関する情報を保持するオブジェクトです。このオブジェクトのプロパティに値を設定することにより、要素のスタイルをダイナミックに変更することができます。要素の指定方法は「エレメント」を参照してください。各プロパティの詳細は「とほほのスタイルシート入門」を参照してください。

HTML
<script>
function func() {
  document.all.xyz.style.color = "red";
}
</script>
<div id="xyz">ABC</div>
<button onclick="func()">OK</button>
