MS-DOSプロンプトで使用可能なツール

LHA.EXE(v2.13/v2.55b)

★ [MS-DOS] フリー
http://www.clione.ne.jp/lha/（休載中）
吉崎栄秦氏開発のフリーソフト。日本では一番よく用いられているLHA形式の元祖ツール。.LZHファイルを圧縮・解凍する。MS-DOSプロンプトから実行する。ショートファイル名（8+3形式）しか扱えないので、今となってはあまりオススメはできないが、元祖ツールには敬意を表したい。(2000.12.17更新)

複数ファイルの圧縮方法：LHA.EXE a XXX.LZH FILE1 FILE2
ディレクトリの圧縮方法：LHA.EXE a -rx XXX.LZH XXXX\*.*
圧縮ファイルの閲覧方法：LHA.EXE t XXX.LZH
圧縮ファイルの解凍方法：LHA.EXE x XXX.LZH
自己解凍形式の作成方法：LHA.EXE s XXX.LZH

LHA32.EXE(v2.67.00)

★ [MS-DOS] フリー
http://www.vector.co.jp/soft/win95/util/se347175.html
吉崎氏が LHA.EXE を Windous 95 に移植したもの。ロングファイル名に対応可能になった。(2005.1.23更新)

LHA32.EXE(v1.04)

★ [MS-DOS] フリー
http://www.clione.ne.jp/lha/（休載中）
take氏作成の同名コマンド。UNLHA32.DLLが必要。(2000.12.17更新)

Zip(v2.3) / Unzip(v5.41)

★ [MS-DOS][OS/2][...] フリー
http://www.info-zip.org/
世界で一番よく用いられている形式、.ZIPファイルを圧縮・解凍する。Info-Zipによるフリーソフト。 WinZipも実はこのコードを利用している。ロングファイルネームにも対応している上、国内、海外、Win, Mac, UNIXいずれでも解凍できるのでおススメ。(2000.12.17更新)

PKZIP.EXE / PKUNZIP.EXE

★ [MS-DOS] シェアウェア
http://www.pkware.com/
ZIP圧縮用のシェアウェア。PKZIP 2.50 for DOS($47) と、PKZIP 4.0 for Windows(CD-ROM:$39.95)がある。これを使うくらいなら、ZIP.EXEのほうがいいと思う。(2000.12.17更新)

Windowsで使用可能なツール

Lhasa v0.17

★★ [Win] フリー
http://www.digitalpad.co.jp/~takechin/
たけちん氏開発のフリーソフト。.LZHと.ZIPファイルを解凍するだけのシンプルな機能。DLLが不要、ダブルクリック一発で解凍というシンプルな設計が心地よい。後発の +Lhaca や 解凍レンジ のお手本ともなった名作。(2000.12.17更新)

+Lhaca Version 0.72

★★★ [Win] フリー
http://sapporo.cool.ne.jp/murayama/Lhaca/
村山富男氏開発のフリーソフト。Lhasa の流れをくむ強化版。DLL不要で圧縮機能がある点がLhasaに勝る。(2000.12.17更新)

解凍レンジ Ver1.41

★★★ [Win] フリー
http://sira.jp/soft/
白川泰洋氏開発のフリーソフト。Lhasaの流れをくむ強化版。DLL不要でARJ、CAB なども解凍できる点が Lhasa に勝る。(2004.2.29更新)

WinZip 8.0

★ [Win] シェアウェア
http://www.winzip.com/
.ZIPファイルなどを圧縮・解凍するシェアウェア。http://www.panda.co.jp/で送金代行サービスと日本語マニュアルを得られるらしい。(2000.12.17更新)

UNLHA.DLL / UNLHA32.DLL(Ver1.53h)

★★ [Win] フリー
http://www.csdinc.co.jp/archiver/
UNLHA.DLLは、Micco氏がLHA.EXEの機能をWindows 3.1用にDLL化したもの。UNLHA32.DLLはこれのWin32版。APIが公開されており、これを用いていろいろなソフトが圧縮・解凍を内部で処理できるようになった。(2000.12.17更新)

UNZIP.DLL / UNZIP32.DLL(Ver5.40)

★★ [Win]
http://www.csdinc.co.jp/archiver/
UNZIP.DLLは、ZIPの解凍機能をshoda T.氏がDLL化したもの。UNZIP32.DLLはこれのWin32版。APIが公開されており、これを用いていろいろなソフトが圧縮・解凍を内部で処理できるようになった。基本的にはフリーだが、企業での使用にはライセンスが必要。(2000.12.17更新)

StuffItシリーズ

★★★ [Mac] シェアウェア / フリー
http://www.act2.co.jp/
Aladdin Systems社のシェアウェアやフリーウェア。Macの世界では定番の圧縮・解凍ツール。StuffIt Deluxeは製品版で、StuffIt形式(.sit)をはじめ、多くの圧縮・解凍をサポート。(LHAには最近のバージョンで対応？？？？)StuffIt Liteはシェアウェア版でDeluxeのサブセット。StuffIt Expanderはフリーソフトで解凍専用。Windows版もある。DropStuffはシェアウェアで圧縮専用。(2000.12.17更新)

MacZip Ver1.06

★★★ [Mac] フリー
http://www.sitec.net/maczip/
MacでZIPファイルの圧縮・解凍が可能なフリーソフト。英語版。(2001.3.25更新)
