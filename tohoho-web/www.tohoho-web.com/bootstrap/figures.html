フィギュア

.figure, .figure-img, .figure-caption はそれぞれ、<figure>, <img>, <figcaption> に対して使用し、図表、図表のイメージ、図表のキャプションを表示します。<img> をレスポンシブにするには .img-fluid を指定します。

Sample
Code
<figure class="figure">
  <img src="100x100.png" class="figure-img img-fluid" alt="100x100 image">
  <figcaption class="figure-caption">Sample of figure</figcaption>
</figure>
