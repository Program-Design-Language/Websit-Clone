イメージ置換

.text-hide はテキストを非表示にします。文字を背景画像として表示するけれども、SEO の対策から文字も指定したい場合、文字を非表示にして埋め込む際に利用されます。

Sample

Image replacement

Sample
<style>
h1 {
  background: url(image_replacement.png);
  width: 284px;
  height: 34px;
}
</style>

<h1 class="text-hide">Image replacement</h1>
