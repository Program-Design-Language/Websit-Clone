クローズアイコン

.close, &times; でクローズアイコンを表示します。スクリーンリーダに読み上げられないように aria-hidden="true" を指定します。

Sample
Sample
<button type="button" class="close" aria-label="Close">
  <span aria-hidden="true">&times;</span>
</button>
