オーバーフロー

.overflow-auto, .overflow-hidden は、オーバーフローしたコンテンツに対してスクロールバーを表示するか、隠すかを指定します。

Sample
あああああああああああああああああああああああああああああああああああああああ
あああああああああああああああああああああああああああああああああああああああ
Code
<div class="overflow-auto" style="width:100px; height:100px;">
  あああああああああああああああああああああああああああああああああああああああ
</div>
<div class="overflow-hidden" style="width:100px; height:100px;">
  あああああああああああああああああああああああああああああああああああああああ
</div>
