クリアフィックス

.float-left や .float-right を使用した場合、そのままではテキストの回り込みが継続してしまうため、親要素に対して .clearfix を指定して、.float-* の効力をクリアします。

Sample
Next text is ...
Code
<div class="clearfix">
  <button type="button" class="btn btn-secondary float-right">float-right</button>
</div>
