環境変数

Rubyは、下記の環境変数を参照します。

環境変数説明
RUBYOPTrubyコマンドに渡すオプションを指定します。(例：-Ke -rkconv）
RUBYPATHRubyスクリプトの探索パスを指定します。環境変数 PATH よりも優先されます。オプション -S 指定時のみ有効です。
RUBYLIBRubyライブラリの探索パスを指定します。特殊変数 $: よりも優先されます。
RUBYLIB_PREFIXDJGPP版/Cygwin版/mswin32版/mingw32版でのみ有効です。path1;path2 または path1 path2 を指定すると、ライブラリ探索パス $: の先頭が path1 にマッチした場合にこれを path2 に置換します。（例：/usr/local/lib/ruby;D:/ruby）
RUBYSHELLOS2版/mswin32版/mingw32版でのみ有効です。Kernel.#system を実行する際のシェルを指定します。省略時は環境変数 COMSPEC が使用されます。
PATHコマンド実行時の探索パスを指定します。未設定の場合は /usr/local/bin:/usr/ucb:/usr/bin:/bin:. が探索されます。
