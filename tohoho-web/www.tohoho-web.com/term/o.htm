◆ ODBC(Open DataBase Connectivity)

Microsoftが定めたデータベースアクセス標準。

◆ Opera

ノルウェーのOpera Software ASA社が開発するWebブラウザを中心とするソフトウェア群。対応するプラットフォームが多く、Windows、Machintosh、Linuxをはじめ、Wiiなどのゲーム機、携帯電話、スマートフォンなどでも利用されている。

◆ ORB(Object Request Broker)

オブジェクトに対する要求を転送するモジュール。オブジェクトの物理的位置を意識しなくてもよいようにするのがORBの役割。

◆ OS(Operating System)

オーエス。コンピュータの基本ソフト。コンピュータではまず、ハードウェアの上でOSが動き、その上でワープロやブラウザなどのアプリケーションソフトが動いている。Windows、Mac OS X、Linux、UNIX、iOS、Androidなど。
