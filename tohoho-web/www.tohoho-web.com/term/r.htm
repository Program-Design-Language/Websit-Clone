◆ RDF(Resource Description Framework)

W3Cが勧告するメタデータ仕様。メタデータ（インデックスデータ）の記述方法や交換方法を規定する。

◆ RealAudioファイル

ウェブでよく用いられる音声ファイル形式。→「とほほのマルチメディア入門」

◆ RealPlayer

RealAudioファイルを再生するプラグインソフト。

◆ REXX(REstructured eXtended eXecutor)

プログラミング言語のひとつ。IBMが開発。レックスと読む。

◆ RFC(Request For Comments)

インターネットに関する標準を定めている文書。RFC1234のように、すべて連続した番号で管理されている。

◆ RGB(Red Green Blue)

光の三原色。赤・緑・青。コンピュータによる色はこの3色を組み合わせてつくる。→「とほほの色入門・色見本」

◆ Ruby

まつものゆきひろ氏が開発したプログラミング言語。PHPと同様、主にサーバサイドで実行されるインタプリタ方式言語。数値や文字列もすべてオブジェクトとして実装されている純粋なオブジェクト指向言語。Perl(Pearl)が6月の誕生石であることから、まつもと氏の誕生石であるルビー(ruby)から名づけられた。
