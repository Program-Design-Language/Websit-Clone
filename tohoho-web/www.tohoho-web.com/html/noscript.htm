<noscript> - スクリプト未対応ブラウザへの配慮

トップ > HTMLリファレンス > <noscript>

概要

形式
<noscript>～</noscript>
サポート
LS / H4 / e3 / Ch1 / Fx1 / Sa1 / Op5.1 / N3
カテゴリ
メタデータコンテンツ
フローコンテンツ
フレージングコンテンツ
親要素
head要素, フレージングコンテンツ を子要素に持てるもの
　※ ただし、noscript要素を先祖に持たない
子要素
head要素内に記述する場合は、0個以上の link, style, meta要素
head要素外に記述する場合は、トランスペアレント
　※ ただし、noscript要素を子孫に持たない
タグの省略
開始タグ:必須 / 終了タグ:必須
属性
グローバル属性

説明

<noscript> は、<script> に対応していないブラウザへのメッセージを記述する際に用います。スクリプト対応ブラウザは <noscript>～</noscript> の間の文書を無視し、未対応ブラウザは <noscript>～</noscript> の間の文書を表示します。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
<script>
document.write("スクリプト対応ブラウザへの記述");
</script>
<noscript>
 <p>スクリプト未対応ブラウザへのメッセージ</p>
</noscript>
