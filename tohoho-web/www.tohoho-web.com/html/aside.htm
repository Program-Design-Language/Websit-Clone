<aside>～</aside>

グローバル属性

説明

HTML5 で追加されました。aside は「余談」や「挿話」を意味します。aside 要素は、周りの要素とすこしだけ関連するけれども、本文とは独立したセクションを示します。活版印刷ではサイドバーとも呼ばれることがあります。広告や、nav要素の集合など、ページの主要コンテンツとは区別されるリード文やサイドバーなどで使用されます。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
<aside>
 <p>余談ですが・・・。</p>
</aside>
表示

余談ですが・・・。
