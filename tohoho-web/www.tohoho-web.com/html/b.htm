説明

b は太字(Bold)の略です。大半のブラウザでは、<b>～</b> で囲まれたテキストが太字で表示されます。

見栄えを定義する要素として、HTML4.01 では「非推奨(deprecated)」とされましたが、HTML5 では「特に付加的な重要性を与えるわけでも、異なるトーンや雰囲気を表現するわけでもないが、実利的な理由から注意を引きつける箇所」を示す要素として、非推奨が解除されました。

ただし、見出しであれば <h1>～<h6>、強調であれば <em>、重要項目であれば <strong> で表現するなど、<b> は最後の手段とすべきと定められています。

単に太字にする目的であれば、CSS の font-weight を使用してください。
