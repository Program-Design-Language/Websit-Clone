説明

u は Under-line(下線)の略です。多くのブラウザでは、<u>～</u> で囲まれたテキストに下線がひかれます。スタイルシートでは <span style="text-decoration:underline">～</span> で代用できます。

見栄えを指定する要素として、HTML 4.01 Tranditional には採用されたものの、HTML 4.01 Strict ではサポート対象外とされましたが、HTML5 では、「中国語テキスト中の固有名詞や、スペルミスの箇所など、本来とは異なった表記を伴うテキストの範囲を示す」として残されました。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
あいうえお <u>かきくけこ</u> さしすせそ
表示
あいうえお かきくけこ さしすせそ
