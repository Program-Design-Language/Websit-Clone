<option> - フォーム部品：オプション

トップ > HTMLリファレンス > <option>

概要

形式
<option>～</option>
サポート
LS / H2 / e2 / Ch1 / Fx1 / Sa1 / Op1 / N2 / i1
カテゴリ
該当無し
親要素
select, datalist, optgroup要素
子要素
label 属性があり、value 属性がある場合: 空。
label 属性があり、value 属性がない場合: テキスト。
label 属性がなく、datalist 要素の子要素でない場合: テキスト (ただし、ホワイトスペースのみは不可)
label 属性がなく、datalist 要素の子要素である場合: テキスト
タグの省略
開始タグ: 必須
終了タグ: 他の option や、optgroup 要素が続く場合、または、この要素が親要素に対する最後の要素の場合、省略可。
属性
グローバル属性
disabled
label
selected
value

説明

<select> による選択部品の選択項目を指定します。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
disabled
LS/H4/e8/Ch/Fx/Sa/Op/N6
この選択項目を選択不可にします。
label=label
LS/H4/e7/Ch/Sa/Op
選択項目のラベルを指定します。これは、<option>～</option> で囲まれたテキストの代わりに表示されます。
selected
LS/H2/e2/Ch/Fx/Sa/Op/N2/i1
この選択項目を選択状態にします。
value=value
LS/H2/e2/Ch/Fx/Sa/Op/N2/i1
フォームパラメータとして送信される、この選択項目の値を指定します。省略した場合は、<option>～</option> で囲まれた文字がそのまま送信されます。

使用例

<select> を参照してください。

空オプション

HTML5 では <option>～</option> の間のテキストには、0個以上の空白文字(空白文字、タブ文字、改行)のみからなるテキストを指定することができませんでしたが、HTML 5.1 ではこれが許可されるようになりました。

HTML
<option></option>
<option>HTML5</option>
<option>HTML 5.1</option>

関連項目

<optgroup>、 <select>、 <datalist>
