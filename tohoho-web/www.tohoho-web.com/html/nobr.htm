<nobr> - 改行禁止

トップ > HTMLリファレンス > <nobr>

概要

形式
<nobr>～</nobr>
サポート
[非推奨] / e2 / Ch1 / Fx1 / Sa1 / Op1 / N2
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
インライン要素
属性
グローバル属性

説明

nobr は NO BReak の略で、改行禁止という意味を持ちます。通常のテキストはブラウザの横幅にあわせて自動的に改行されてしまいますが、<nobr>～</nobr> で囲まれたテキストは改行されません。HTML の標準には取り入れられていません。代わりにスタイルシートの white-space を参照してください。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
<nobr>
ブラウザの横幅が狭くてもこの文章は改行されません。
</nobr>
表示
ブラウザの横幅が狭くてもこの文章は改行されません。

関連項目

<nobr> は見栄えを定義するタグとして、W3C には採用されていません。スタイルシートで改行禁止を実現するには、white-space で nowrap を指定します。

HTML
<div style="white-space:nowrap;">
ブラウザの横幅が狭くてもこの文章は改行されません。
</div>

関連項目

<br>、 <wbr>
