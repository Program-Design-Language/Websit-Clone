<spacer> - スペーサー

トップ > HTMLリファレンス > <spacer>

概要

形式
<spacer type="...">
サポート
[非推奨] / N3-N4
タグの省略
開始タグ：必須、終了タグ：無し
属性
グローバル属性

説明

Netscape Navigator 3.0～4.0 のみでサポートされたタグで、指定した大きさのスペースを挿入します。Internet Explorer ではサポートされず、スタイルシートが普及していない当時は、1×1ピクセルの画像を width, height 指定で貼り付けてスペースを制御するテクニックが使用されていました。

現在では、空白を制御するには、margin, padding などのスタイルシートを利用してください。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
type=type
[非推奨] N3
スペースの形を指定します。
horizontal
横方向に size ピクセル分
vertical
縦方向に size ピクセル分
block
width × height ピクセル分
size=n
[非推奨] N3
スペースの縦幅や横幅をピクセルで指定します。
height=n
[非推奨] N3
スペースの高さをピクセルで指定します。
width=n
[非推奨] N3
縦幅をピクセルで指定します。
align=align
[非推奨] N3
周りのテキストとの位置関係を指定します。

使用例

HTML
<spacer type=horizontal size=20>あいうえお<br>
<spacer type=horizontal size=30>かきくけこ<br>
<spacer type=horizontal size=40>さしすせそ
表示
　　あいうえお
　　　かきくけこ
　　　　さしすせそ
