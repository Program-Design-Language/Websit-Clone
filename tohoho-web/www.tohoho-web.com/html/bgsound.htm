概要

形式
<bgsound src="...">
サポート
[非推奨] / e2
タグの省略
開始タグ：必須、終了タグ：無し
属性
グローバル属性
(その他下記参照)

説明

bgsound は BackGround SOUND の略です。Internet Explorer 独自の機能で、ページが開かれた際にバックグランドに音楽(BGM)を再生します。

<bgsound> はHTMLの標準には採用されていません。代わりに <audio> を使用してください。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
balance=balance
[非推奨] e4
左右の音量のバランスを -10000～10000 の範囲で指定します。
loop=loop
[非推奨] e2
再生する回数を指定します。infinite または -1 を指定すると無限に再生を繰り返します。
src=url
[非推奨] e2
サウンドファイルを指定します。*.wav, *.au, *.mid, *.aif などのサウンド形式をサポートしています。
volume=volume
[非推奨] e4
音量を -10000 から 0 までの範囲で指定します。0 が最大音量です。

使用例

HTML
<html>
 <head>
  <title>タイトル</title>
  <bgsound loop="infinite" src="sound/xxx.mid">
 </head>
 <body>
