<nolayer> - レイヤ未対応ブラウザへの配慮

トップ > HTMLリファレンス > <nolayer>

概要

形式
<nolayer>～</nolayer>
サポート
[非推奨] / N4
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
ブロック要素 / インライン要素
属性
グローバル属性

説明

<nolayer> で、レイヤに対応していないブラウザに対するメッセージを記述します。Netscape Communicator 4.0 でサポートされましたが、Netscape 6.0 でも廃止されました。HTML の標準にも取り入れられていません。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
<layer src="demo.html"></layer>
<nolayer>このデモはレイヤ対応ブラウザで見てください。</nolayer>
