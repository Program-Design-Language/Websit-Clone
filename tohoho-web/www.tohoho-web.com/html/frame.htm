<frame> - フレーム

トップ > HTMLリファレンス > <frame>

概要

形式
<frame src="...">
サポート
[非推奨] / H4-4 / e3 / Ch1 / Fx1 / Sa1 / Op1 / N2
タグの省略
開始タグ：必須、終了タグ：無し
属性
グローバル属性
(その他下記参照)

説明

<frameset> の中で使用し、個々のフレームを表示します。詳細は <frameset> を参照してください。

属性

【共通属性】
グローバル属性
詳細は グローバル属性 を参照してください。
【重要属性】
src=url
[非推奨] H4/e/Ch/Fx/Sa/Op/N2
フレームの中身に表示する文書の URL を指定します。
【表示関連】
height=n
[非推奨] e4-?
フレームの高さをピクセルまたはパーセンテージで指定します。
width=n
[非推奨] e4-?
フレームの横幅をピクセルまたはパーセンテージで指定します。
bordercolor=color
[非推奨] e4/Fx/N3
フレームの境界線の色を指定します。
frameborder=frameborder
[非推奨] H4/e3/Ch/Fx/Sa/Op/N3
境界線を表示する場合は 1 を、表示しない場合は 0 を指定します。Internet Explorer や Netscapeブラウザでは yes と no も使用可能です。境界線の両側で 0 が指定されると、境界線が表示されなくなります。
marginheight=n
[非推奨] H4/e3/Ch/Fx/Sa/Op/N2
フレームの上下の余白をピクセル単位で指定します。
marginwidth=n
[非推奨] H4/e3/Ch/Fx/Sa/Op/N2
フレームの左右の余白をピクセル単位で指定します。
noresize
[非推奨] H4/e3/Ch/Fx/Sa/Op/N2
フレームのリサイズを禁止します。
【その他の属性】
longdesc=url
[非推奨] H4
このフレームに関する説明が title 属性で示せないくらい長文の場合に、その説明ページの URL を指定します。
scrolling=scrolling
[非推奨] H4/e3/Ch/Fx/Sa/Op/N2
フレームのスクロールバーの表示を制御します。
yes
常に表示
no
常に非表示
auto
自動(既定値)
allowtransparency=bool
[非推奨] e5.5/Fx
フレームを透明色にする場合に true を指定します。既定値は false です。透明にした場合、<frameset> を記述した HTMLファイル側に設定された背景が表示されます。Internet Explorer ではこの属性を指定したフレームのみが透明になりますが、Firefox では、すべてのフレームが透明になってしまうようです(Firefox 34で確認)。
