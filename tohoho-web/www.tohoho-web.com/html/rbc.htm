説明

rbc は Ruby Base container の略です。ルビを振る対象を指定します。当初のルビの勧告 (http://www.w3.org/TR/ruby) には定義されていましたが、HTML5 や HTML Living Standard には採用されませんでした。詳細は <ruby>を参照してください。
