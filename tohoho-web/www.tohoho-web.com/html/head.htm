<head> - ヘッダ

トップ > HTMLリファレンス > <head>

概要

形式
<head>～</head>
サポート
LS / H2 / e2 / Ch1 / Fx1 / Sa1 / Op1 / N2 / i1
カテゴリ
該当なし
親要素
html要素 (最初の子要素として)
子要素
1個の title 要素と、0個または 1個の base 要素を含む メタデータコンテンツ。ただし、ドキュメントが iframe の srcdoc で指定されたものである場合、もしくは、title 情報が上位のプロトコルで指定された場合は、title 要素は 0 個でもよい。
タグの省略
開始タグ: head 要素が空の場合、または、head 要素内の最初のコンテンツが要素の場合、省略可能。
終了タグ: head 要素の直後が ASCII ホワイトスペースやコメントでなければ、省略可能。
属性
グローバル属性

説明

この文書のヘッダ部を指定します。<head>～</head> で囲んだ部分がヘッダ部になります。ヘッダ部には、この文書全体に関する指定などを書きます。

開始タグ、終了タグ共に省略可能ですが、明示的に書いておいた方がわかり易いでしょう。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
profile=url
[非推奨] H4-4/N6
メタデータプロファイルの URL を指定します。<meta> タグで指定した author や keywords などの情報を解釈する際などに役立てられると定義されていますが、プロファイルのデータ形式も定義されておらず、実際はほとんど使用されていません。HTML4.0 で定義されましたが、HTML5 では廃止されました。

使用例

<html> を参照してください。
