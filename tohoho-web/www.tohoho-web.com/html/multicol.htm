<multicol> - 段組み

トップ > HTMLリファレンス > <multicol>

概要

形式
<multicol cols="...">～</multicol>
サポート
[非推奨] / N3-N4
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
ブロック要素 / インライン要素
属性
グローバル属性

説明

multicol は MULTI COLumn の略です。Netscape Navigator 3.0～4.* のみでサポートされていたタグで、文書を 2段組や 3段組などで表示します。段組を行う際には非常に便利な機能でしたが、見栄えを定義するものとして W3C には採用されず、Netscape 6.0 でも廃止されました。現在では、CSS の columns でマルチカラムを実現できます。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
cols=n
[非推奨] N3-N4
段組の段数を指定します。
gutter=gutter
[非推奨] N3-N4
段組の間の隙間の幅をピクセルで指定します。省略時は 10 になります。
width=n
[非推奨] N3-N4
ひとつの段の横幅をピクセルで指定します。
