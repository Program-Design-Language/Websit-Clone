説明

<blockquote>～</blockquote> で囲まれた文章が、引用文であることを明示します。

スタイルシートが普及する以前はインデントを実現する手段が他に無かったため、<blockquote> はインデントの代役として使用された時期もありました。Microsoft の HTML エディタでも、インデントを指定すると <blockquote> を書き出していましたし、このサイトでもインデントの代わりに使用していましたが、本来の意味は「引用」です。スタイルシートを問題なく利用可能になった現在では、インデントの目的にはスタイルシートの margin-right や margin-left を用いてください。

使用例

HTML
いまから引用を開始します。
<blockquote>
 この部分が引用文です。
 多くのブラウザでは、引用文の前後左右に隙間があきます。
</blockquote>
引用が終わりました。
表示
