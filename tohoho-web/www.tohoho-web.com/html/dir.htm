<dir> - ディレクトリリスト

トップ > HTMLリファレンス > <dir>

概要

形式
<dir>～</dir>
サポート
[非推奨] / H2-H4T / e2 / Ch1 / Fx1 / Sa1 / Op1 / i1
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
li
属性
グローバル属性
(その他下記参照)

説明

dir は DIRectory list の略です。<menu> と同様、リストを表示しますが、あまり使用されていません。HTML4.01 では非推奨として定義されており、HTML5 では廃止されました。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
compact
[非推奨] H2-H4T
リストを極力コンパクトな形式で表示します。
type=type
[非推奨] e/Fx/i4
リストの行頭のマークの形を指定します。
disc
黒丸
circle
白丸
square
四角

使用例

HTML
<dir>
 <li>ファイルＡ
 <li>ファイルＢ
</dir>

関連項目

<menu>、 <ul>、 <li>
