説明

bdo は、i18n BiDirectional Over-ride の略です。i18nというのは、intarnationalization(国際化)のことで、i と n の間に 18個のアルファベットがあることから i18n と呼ばれています。dir 属性は必須で、ltr または rtl を指定します。

英語など左→右方向に記述する言語と、ヘブライ語など右→左方向に書く言語を混在させる場合に用いますが、単純に文字を右から左に表示したいときには <html> や <div> や <span> などの dir 属性を用います。<bdo> を用いるのはごく稀なケースのようです。詳細は dir 属性を参照してください。
