説明

<pre> と同様、ソースコード中のスペースや改行をそのまま表示します。< や > や & もそのまま表示されます。

Internet Explorer 2.0 以前、および Netscape 6 では <plaintext>～</plaintext> までをべたテキストで表示していましたが、Internet Explorer 3.0 以降、Netscape Communicator 4 以前、Chrome, Firefox, Safari, Opera などでは、<plaintext> 以降ファイルの終わりまでをすべてべたテキストとして表示します。

この要素は、HTML2.0 や HTML3.2 では非推奨とされていました。HTML4.01 で廃止されました。代わりに <pre> を使用してください。
