<big> - 大きな文字

トップ > HTMLリファレンス > <big>

概要

形式
<big>～</big>
サポート
[非推奨] / H3-H4 / e3 / Ch1 / Fx1 / Sa1 / Op1 / N2
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
インライン要素
属性
グローバル属性

説明

<big>～</big> で囲まれたテキストをひとまわり大きくします。<big> タグを多重に(入れ子に)使用することも可能です。

HTML4.01 Strictでも使用可能でしたが、HTML5 では廃止されました。代わりにスタイルシートの font-size を使用してください。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
普通の大きさのフォントにくらべて、<br>
<big>大きな<big>大きな</big>フォントに</big>なります。
表示
普通の大きさのフォントにくらべて、
大きな大きなフォントになります。

関連項目

<i>、 <b>、 <tt>、 <small>
