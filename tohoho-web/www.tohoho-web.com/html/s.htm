説明

s は Strike-through(削除する)の略です。<s>～</s> で囲まれたテキストが削除されたものであることを示すために、打ち消し線がひかれます。

HTML4.01 では削除予定として定義されていましたが、HTML5 では、<s> は 「すでに正確ではない、もしくは関連が無いコンテンツを示す」 として正式に定義されています。ただし、編集上 「削除された」 の意味であれば、<del> を使用することが推奨されています。

属性

グローバル属性
詳細は グローバル属性 を参照してください。

使用例

HTML
<p>レモネード発売中</p>
<p><s>1杯300円</s></p>
<p>値下げしました。1杯250円!!</p>
表示

レモネード発売中

1杯300円

値下げしました。1杯250円!!
