<menuitem> - メニューアイテム

トップ > HTMLリファレンス > <menuitem>

概要

形式
<menuitem label="...">
サポート
[非推奨] / H5.1-5.1 / Fx8
カテゴリ
なし
親要素
menu(type="context")
子要素
なし
タグの省略
開始タグ:必須 / 終了タグ:なし
属性
グローバル属性

説明

HTML 5.1 で追加されました。<menu> と組み合わせてコンテキストメニュー(ポップアップメニュー)を実現します。

HTML Living Standard には採用されていません。また、あまり実装されなかったことを理由に、HTML 5.2 でも廃止されました。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
type=type
[非推奨] H5.1-5.1
コマンドタイプを指定します。
label=label
[非推奨] H5.1-5.1
ユーザに表示するラベルを指定します。
icon=icon
[非推奨] H5.1-5.1
アイコンを指定します。
disabled
[非推奨] H5.1-5.1
メニューアイテムを無効にします。
checked
[非推奨] H5.1-5.1
メニューアイテムをチェック状態にします。
radiogroup=radiogroup
[非推奨] H5.1-5.1
ラジオグループを指定します。
default
[非推奨] H5.1-5.1
デフォルトメニューに指定します。
