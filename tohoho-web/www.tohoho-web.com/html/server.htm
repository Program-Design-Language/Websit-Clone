<server> - サーバーサイド JavaScript

トップ > HTMLリファレンス > <server>

概要

形式
<server>～</server>
サポート
[非推奨] / Netscape server
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
文字列
属性
なし

説明

<server> タグは Server Side JavaScript を実現するためのタグです。

Netscape 社のウェブサーバーがサポートしていた機能で、<server>～</server> で囲まれた部分に記述された JavaScript を、Webサーバーが解釈してその結果をブラウザに表示します。SSI や ASP の動作と似ています。

現在ではほとんど使用されていません。

属性

なし

使用例

HTML
あなたのIPアドレスは
<server>
write(request.ip);
</server>
です。
表示
あなたのIPアドレスは 192.168.1.23 です。
