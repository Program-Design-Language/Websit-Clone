language=language - スクリプト言語指定

トップ > HTMLリファレンス > language=language

属性

<タグ名 language=language> - スクリプト言語指定

説明

Internet Explorer 4.0 以降のみで有効な属性で、onClick などのイベントハンドラで実行するスクリプトの言語を指定します。

  • language="JavaScript" → JavaScript
  • language="JScript" → JavaScript
  • language="VBScript" → VBScript
  • language="VBS" → VBScript

使用例

HTML
<form action="#">
 <input type="button" value="JS" language="JavaScript"
  onclick="alert('これは JavaScript です。')">
 <input type="button" value="VB" language="VBScript"
  onclick="MsgBox('これは VBScript です。')">
</form>
