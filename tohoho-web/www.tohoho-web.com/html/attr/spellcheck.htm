spellcheck= - スペルチェック

トップ > HTMLリファレンス > spellcheck=

属性

<タグ名 spellcheck=true|false> - スペルチェック

説明

HTML5 でサポート予定の属性で、textarea などにユーザが入力した文章に対してスペルチェックを行うかどうかを true または false で指定します。スペルチェック結果の表示方法はブラウザにまかされます。例えば Opera では、単語の下に赤い波線が表示されます。

使用例

HTML
<textarea spellcheck="true">This is a boook.</textarea>
表示
This is a boook.
