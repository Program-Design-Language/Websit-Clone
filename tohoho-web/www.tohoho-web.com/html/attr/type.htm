type=type - MIMEタイプ指定

トップ > HTMLリファレンス > type=type

属性

<タグ名 type=type> - MIMEタイプ指定

説明

<a>, <area>, <link>, <embed>, <object>, <script>, <source>, <style> 等のタグで、MIMEタイプを指定します。MIMEタイプには次のようなものがあります。

MIMEタイプ説明
text/htmlHTML文書。
text/plainべたテキスト文書。
image/gifGIF画像。
image/jpegJPEG画像。
text/javascriptJavaScriptスクリプト。
text/cssスタイルシートファイル(CSS)。
