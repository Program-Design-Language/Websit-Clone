data-*= - ユーザ拡張属性

トップ > HTMLリファレンス > data-*=

属性

<タグ名 data-*=value> - ユーザ拡張属性

説明

HTML5 でサポートされた仕様で、* の部分に好きな名前を指定し、ユーザが独自の属性を追加することができます。

使用例

HTML
<div data-insdate="2011-05-29">～です。(2011年5月29日追記)</div>
