hidefocus=true - フォーカスを隠す

トップ > HTMLリファレンス > hidefocus=true

属性

<タグ名 hidefocus=true> - フォーカスを隠す

説明

Internet Explorer 5.5 でサポートされた属性です。通常は、ボタンやリンクにフォーカスがあたると点線で囲まれますが、hidefocus=true を指定すると点線が表示されません。Internet Explorer 5.5 以降と Edge でサポートされていますが、Chrome、Firefox、Opera、Safari ではサポートされていません。

使用例

HTML
<button>OK</button><br>
<button hidefocus="true">OK</button><br>
<a href="xxx.htm">XXX</a><br>
<a href="xxx.htm" hidefocus="true">XXX</a><br>
