ping= - ping通知

トップ > HTMLリファレンス > ping=

属性

<タグ名 ping=url> - ping通知

説明

HTML5 でサポートを検討していましたが、最終案では不採用となりました。ping属性が指定されていると、そのリンクがクリックされた際に、ping属性で指定されたURLを呼び出します。これは、リンクが押されたことのロギングをとるなどの目的で使用されます。

使用例

HTML
<a href="http://www.tohoho-web.com/html/a.htm" ping="http://www.tohoho-web.com/log?a.htm">a</a>
