char=char - 文字による位置揃え

トップ > HTMLリファレンス > char=char

属性

<タグ名 char=char> - 文字による位置揃え

説明

char 属性は、数値を小数点の位置で揃えて表示したい場合などに用います。char="." のように、揃えたい文字を 1文字指定します。省略時の値は、その言語における小数点(英語ではピリオド "."、フランス語ではカンマ ",")です。

この属性は、align="char" とした場合に有効です。

HTML
<table border=1>
<colgroup align="center">
<colgroup align="char" char=".">
<tr><td>田中</td><td>123.456</td></tr>
<tr><td>鈴木</td><td>1.2345678</td></tr>
</table>

<colgroup>, <col>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td> で指定可能と定義されていますが、サポートしているブラウザはまだ少ないようです。
