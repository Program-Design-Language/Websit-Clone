onbeforeunload=script - ページアンロード前のハンドラ

トップ > HTMLリファレンス > onbeforeunload=script

属性

<タグ名 onbeforeunload=script>

説明

ページがアンロードされる前に呼ばれ、ページを離れてもよいか確認するダイアログが表示されます。ダイアログに表示するメッセージを return で返します。Firefox ではメッセージが表示されないようです。(Firefox 34で確認)

サンプル

HTML
<body onbeforeunload="return 'TEST'">
HTML
<script>
window.onbeforeunload = function(e) {
    return "TEST";
}
</script>
