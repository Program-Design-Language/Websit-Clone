inputmode=mode - 入力モード

トップ > HTMLリファレンス > inputmode=mode

属性

<タグ名 inputmode=mode> - 入力モード

サポート

LS / H5.1-5.1

説明

input や textarea 要素に対する入力モードを指定します。HTML 5.1 や HTML Living Standard では定義されていますが、HTML 5.2 では削除されています。まだ、実装例も少ないようです。スタイルシートの ime-mode も参照してください。

モードサポート説明
noneLS入力モードを指定しない。
textLSテキスト入力モード。
telLS/H5.1-5.1電話番号入力モード。
urlLS/H5.1-5.1URL 入力モード。
emailLS/H5.1-5.1Eメールアドレス入力モード。
numericLS/H5.1-5.1数値入力モード。
decimalLS整数値入力モード。
searchLS検索語入力モード。
verbatimH5.1-5.1ラテン文字逐語モード(HTML 5.1の規定値)
latinH5.1-5.1ラテン文字モード
latin-nameH5.1-5.1ラテン文字入力モード(名前入力用)。
latin-proseH5.1-5.1ラテン文字文章入力モード。
full-width-latinH5.1-5.1全角ラテン文字入力モード
kanaH5.1-5.1漢字変換ひらがな入力モード。
kana-nameH5.1-5.1漢字変換ひらがな入力モード(名前入力用)。
katakanaH5.1-5.1カタカナ入力モード。

使用例

HTML
<div>TEL: <input type="text" inputmode="tel"></div>
