autocapitalize=mode - 自動大文字入力制御

トップ > HTMLリファレンス > autocapitalize=mode

属性

<タグ名 autocapitalize=mode> - 自動大文字入力制御

サポート

Safari(iOS)

説明

input や textarea に対して、iOS の Safari の英語入力モードで入力する際、最初の 1文字目を入力する際に入力パネルが自動的に大文字入力モードになることがあります。autocapitalize は、この機能を制御する際に用います。mode には下記のいずれかを指定します。

off
自動大文字化機能をオフにします。
none
自動大文字化機能をオフにします。
on
自動大文字化機能をオンにします。
sentences
文章毎に自動大文字化機能をオンにします。
words
単語毎に自動大文字化機能をオンにします。
characters
文字毎に自動大文字化機能をオンにします。

使用例

HTML
<input type="text" autocapitalize="off" name="login_id">
