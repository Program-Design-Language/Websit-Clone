rel=relation - 関係指定

トップ > HTMLリファレンス > rel=relation

属性

<タグ名 rel=relation> - 関係指定

説明

<a>, <area>, <link> の rel に指定する値として、HTML5 では下記が定義されています。

値意味
stylesheetスタイルシートファイルを取り込みます。link要素でのみ使用可能です。
prevひとつ前の文書。
nextひとつ次の文書。
alternateこの文書の代替となる文書。
authorこの文書の作者に関する文書。
bookmarkこの文書の作者に関する文書。
help関連するヘルプ文書。
icon関連するアイコンファイル。link要素でのみ使用可能です。
licenseライセンスに関する文書。
nofollowこの文書の作者(作成者)が参照する文書を推薦していないことを示します。a要素、area要素で指定できます。
noreferrerこのリンク先を参照した時、ユーザエージェント(ブラウザ)が、HTTP Refereヘッダを送信しないことを要求します。a要素、area要素で指定できます。
prefetch参照先のリソースを先行してキャッシュすることを示します。
search現在の文書や関連文書を検索するための検索ページを示します。
tagこの文書に適用されるタグを示します。a要素と area要素で使用できます。

使用例

HTML
<link rev="stylesheet" href="css/style.html" type="text/css">
<link rev="prev" href="chapter1.html">
<link rev="next" href="chapter3.html">
<link rev="alternate" href="simple-format.html">
<link rev="author" href="author.html">
<link rev="bookmark" href="bookmark.html">
<link rev="help" href="help.html">
<link rev="icon" href="image/favicon.png" type="image-x-icon">
<link rev="license" href="license.html">
