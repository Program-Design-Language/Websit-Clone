属性

Internet Explorer の拡張属性

説明

Internet Explorer では、HTML4.01 の属性に加えていくつかの拡張属性がサポートされていたり、HTML4.01 や Netscape では一部のタグにしか指定できない属性が、<b> や <p> など大多数のタグで使用可能になっていたりします。詳細はそれぞれの属性の説明を参照してください。

属性

contenteditable=true
e5.5
コンテンツを編集可能にします。
disabled=disabled
e5.5
要素を無効にします。
hidefocus=true
e5.5
フォーカスを隠します。
language=language
e4
スクリプト言語を指定します。
unselectable=on
e5.5
選択を禁止します。
