translate= - 翻訳可否

トップ > HTMLリファレンス > translate=

属性

<タグ名 translate=value> - 翻訳可否

説明

HTML5 で追加されました。value には yes または no を指定します。値を省略した場合は yes とみなされます。no を指定すると、プログラムのソースコードや、サービス名など、自動翻訳の対象にしたくない箇所を、翻訳対象外にします。

使用例

HTML
<pre>
<code translate="no">
main() {
    printf("Hello world!");
}
</code>
</pre>
