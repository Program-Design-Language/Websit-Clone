<td> - テーブルデータ

トップ > HTMLリファレンス > <td>

概要

形式
<td>～</td>
サポート
LS / H3 / e2 / Ch1 / Fx1 / Sa1 / Op1 / N2
カテゴリ
セクショニングルート
親要素
tr要素
子要素
フローコンテンツ
タグの省略
開始タグ: 必須
終了タグ: td または th 要素が直後に続く場合、もしくは、親要素に対する最後の要素の場合、省略可能。
属性
グローバル属性
colspan
rowspan
headers

説明

td は Table Data の略です。テーブルのセルの内容を記述します。詳細は <table> を参照してください。

属性

【重要属性】
グローバル属性
詳細は グローバル属性 を参照してください。
colspan=n
LS/H3/e2/N2
セルを n個、横方向に連結します。
rowspan=n
LS/H3/e2/N2
セルを n個、縦方向に連結します。
【ヘッダ用属性】
headers=headers
LS/H4
このセルに対応する見出しセルのリストを、id 属性で指定します。
scope=scope
[非推奨] H4-4
主に <th> で使用します。HTML4.01 では td 要素でも定義されていましたが、HTML5 では th 要素でのみ使用可能となっています。
abbr=abbr
[非推奨] H4-4
主に <th> で使用します。HTML4.01 では td 要素でも定義されていましたが、HTML5 では th 要素でのみ使用可能となっています。
axis=axis
[非推奨] H4-4
主に <th> で使用します。HTML5 では廃止されました。
【ボーダー関連属性】
bordercolor=color
[非推奨] e2/Ch/Fx/Sa/Op/N4
枠線の色を指定します。
bordercolordark=color
[非推奨] e2
立体的な枠線の暗い部分の色を指定します。
bordercolorlight=color
[非推奨] e2
立体的な枠線の明るい部分の色を指定します。
【背景関連属性】
background=url
[非推奨] e3/Ch/Fx/Sa/Op/N4
背景画像を指定します。<table>, <tr>, <th> でも指定可能です。
bgcolor=color
[非推奨] H4T-4T/e2/Ch/Fx/Sa/Op/N3
背景色を指定します。<table>, <tr>, <th> でも指定可能です。
【配置関連属性】
align=align
[非推奨] H3-H4/e2/Ch/Fx/Sa/Op/N2
セル内データの横方向の配置を指定します。HTML5 では廃止予定です。
　　center：中央表示(H3/e2/Ch/Fx/Sa/Op/N2)
　　char：指定文字揃え(H4)
　　justify：均等割付(H4)
　　left：左寄せ(H3/e2/Ch/Fx/Sa/Op/N2)
　　right：右寄せ(H3/e2/Ch/Fx/Sa/Op/N2)
valign=valign
[非推奨] H3-H4/e2/Ch/Fx/Sa/Op/N2
セル内データの縦方向の配置を指定します。HTML5 では廃止予定です。
　　baseline：ベースライン揃え(H4/e2/Ch/Fx/Sa/Op/N2)
　　bottom：下揃え(H3/e2/Ch/Fx/Sa/Op/N2)
　　center：中央揃え(e2/Ch/Fx/Sa/Op/N2)
　　middle：中央揃え(H3/e2/Ch/Fx/Sa/Op/N2)
　　top：上揃え(H3/e2/Ch/Fx/Sa/Op/N2)
height=n
[非推奨] H3-H4T/e2/N2
セルの高さを 300 のようなピクセル値、または 50% のようなパーセントで指定します。HTML5 では廃止予定です。
width=n
[非推奨] H3-H4T/e2/N2
セルの横幅を 300 のようなピクセル値、または 50% のようなパーセントで指定します。HTML5 では廃止予定です。
nowrap
[非推奨] H3-H4T/e2/N2
セル内データが自動改行されないようにします。HTML5 では廃止予定です。
char=char
[非推奨] H4-4
位置揃えする文字を指定します。HTML5 では廃止予定です。
charoff=n
[非推奨] H4-4
char 属性で指定した文字を、セルの左端からどのくらいの位置に表示するかのオフセットを指定します。HTML5 では廃止予定です。
