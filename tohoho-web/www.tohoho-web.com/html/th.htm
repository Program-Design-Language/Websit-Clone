<th> - テーブルヘッダ項目

トップ > HTMLリファレンス > <th>

概要

形式
<th>～</th>
サポート
LS / H3 / e2 / Ch1 / Fx1 / Sa1 / Op1 / N2
カテゴリ
該当無し
親要素
tr 要素
子要素
フローコンテンツ (ただし、header, footer要素, セクショニングコンテンツ, ヘッディングコンテンツ を子孫に持たない)
タグの省略
開始タグ: 必須
終了タグ: td または th 要素が直後に続く場合、もしくは、親要素に対する最後の要素の場合、省略可能。
属性
グローバル属性
colspan
rowspan
headers
scope
abbr

説明

th は Table Header の略です。テーブルのヘッダ部分のセルの内容を記述します。多くのブラウザでは、文字が太字の中央表示で表示されます。

属性

【重要属性】
グローバル属性
詳細は グローバル属性 を参照してください。
colspan=n
LS/H3/e2/N2
セルを n個、横方向に連結します。
rowspan=n
LS/H3/e2/N2
セルを n個、縦方向に連結します。
【ヘッダ用属性】
headers=headers
LS/H4
このセルに対応する見出しセルのリストを、id 属性で指定します。
scope=scope
LS/H4
見出しセルに対して指定します。
row
見出しが列方向に対する見出しであることを示します。
col
見出しが行方向に対する見出しであることを示します。
rowgroup
見出しが列グループに対する見出しであることを示します。
colgroup
見出しが行グループに対する見出しであることを示します。
auto
コンテキストに従い解釈されます。
abbr=abbr
LS/H4
主に <th> で使用します。セルの内容の概要を記述します。音声ブラウザがセルの概略を読み上げるのに役立ちます。
axis=axis
[非推奨] H4-4
多次元配列データのための軸情報などを記述します。HTML5 では廃止されました。
【ボーダー関連属性】
bordercolor=color
[非推奨] e2/Ch/Fx/Sa/Op/N4
枠線の色を指定します。
bordercolordark=color
[非推奨] e2
立体的な枠線の暗い部分の色を指定します。
bordercolorlight=color
[非推奨] e2
立体的な枠線の明るい部分の色を指定します。
【背景関連属性】
background=url
[非推奨] e3/Ch/Fx/Sa/Op/N4
背景画像を指定します。<table>, <tr>, <th> でも指定可能です。
bgcolor=color
[非推奨] H4T-4T/e2/Ch/Fx/Sa/Op/N3
背景色を指定します。<table>, <tr>, <th> でも指定可能です。
【配置関連属性】
align=align
[非推奨] H3-H4/e2/Ch/Fx/Sa/Op/N2
セル内データの横方向の配置を指定します。HTML5 では廃止予定です。
center
中央表示(H3/e2/Ch/Fx/Sa/Op/N2)
char
指定文字揃え(H4)
justify
均等割付(H4)
left
左寄せ(H3/e2/Ch/Fx/Sa/Op/N2)
right
右寄せ(H3/e2/Ch/Fx/Sa/Op/N2)
valign=valign
[非推奨] H3-H4/e2/Ch/Fx/Sa/Op/N2
セル内データの縦方向の配置を指定します。HTML5 では廃止予定です。
baseline
ベースライン揃え(H4/e2/Ch/Fx/Sa/Op/N2)
bottom
下揃え(H3/e2/Ch/Fx/Sa/Op/N2)
center
中央揃え(e2/Ch/Fx/Sa/Op/N2)
middle
中央揃え(H3/e2/Ch/Fx/Sa/Op/N2)
top
上揃え(H3/e2/Ch/Fx/Sa/Op/N2)
height=n
[非推奨] H3-H4T/e2/N2
セルの高さを 300 のようなピクセル値、または 50% のようなパーセントで指定します。HTML5 では廃止予定です。
width=n
[非推奨] H3-H4T/e2/N2
セルの横幅を 300 のようなピクセル値、または 50% のようなパーセントで指定します。HTML5 では廃止予定です。
nowrap
[非推奨] H3-H4T/e2/N2
セル内データが自動改行されないようにします。HTML5 では廃止予定です。
char=char
[非推奨] H4-4
位置揃えする文字を指定します。HTML5 では廃止予定です。
charoff=n
[非推奨] H4-4
char 属性で指定した文字を、セルの左端からどのくらいの位置に表示するかのオフセットを指定します。HTML5 では廃止予定です。
