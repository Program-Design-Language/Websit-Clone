<comment> - コメント

トップ > HTMLリファレンス > <comment>

概要

形式
<comment>～</comment>
サポート
[非推奨] e2-10
タグの省略
開始タグ：必須、終了タグ：必須
包含可能要素
comment 以外のすべて
属性
グローバル属性
(その他下記参照)

説明

<!-- ～ -->と同様、HTMLのソース中にコメント(注釈)を記述します。<comment>～</comment> の部分はブラウザに表示されません。

Internet Explorer 独自機能であり、現在は使用されていません。Internet Explorer 11 ではサポート中止されました。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
id=id
[非推奨] e4-10
IDを指定します。
lang=lang
[非推奨] e4-10
言語を指定します。

使用例

HTML
<comment>
 このページは Internet Explorer で参照してください。
</comment>
