<basefont> - 基本フォントの指定

トップ > HTMLリファレンス > <basefont>

概要

形式
<basefont size="..." color="..." face="...">
サポート
[非推奨] / H3-H4T / e2-9 / N2-4
タグの省略
開始タグ：必須、終了タグ：無し
属性
グローバル属性
(その他下記参照)

説明

basefont は、これ以降のフォントの、基準サイズや基準の色などを指定します。属性についての詳細は <font> を参照してください。

<base> は <head>～</head> の間に記述しますが、<basefont> は <body>～</body> の間に記述します。

HTML3.0, HTML4.01 Transitional で定義されていましたが、HTML5 で廃止されました。Internet Explorer ではサポートされていますが、Chrome, Firefox, Safari, Opera などではサポートされていません。Internet Explorer 10 でもサポートを廃止しました。代わりにスタイルシートの font を使用してください。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
size=size
[非推奨] H3-H4T/e3/N2
フォントサイズを指定します。HTML4.01 では必須属性として定義されています。現在では代わりにスタイルシートの font-size を使用してください。
color=color
[非推奨] H4T/e2
フォントの色を指定します。現在では代わりにスタイルシートの color を使用してください。
face=face
[非推奨] H4T/e2
フォントを指定します。現在では代わりにスタイルシートの font-family を使用してください。

使用例

HTML
<html>
 <head>
  <title>タイトル</title>
 </head>
 <body>
  <basefont size="4">
  これ以降のフォントサイズを 4 にします。<br>
  <basefont size="5">
  これ以降のフォントサイズを 5 にします。<br>
　　　：
表示
これ以降のフォントサイズを 4 にします。
これ以降のフォントサイズを 5 にします。
