<time> - 時刻

トップ > HTMLリファレンス > <time>

概要

形式
<time>～</time>
サポート
LS / H5 / e9 / Ch33 / Fx22 / Op22 / Sa7
カテゴリ
フローコンテンツ
フレージングコンテンツ
パルパブルコンテンツ
親要素
フレージングコンテンツ を子要素に持てるもの
子要素
datetime 属性ありの場合 フレージングコンテンツ、もしくは下記フォーマットのテキスト
タグの省略
開始タグ:必須 / 終了タグ:必須
属性
グローバル属性
datetime

説明

timeは、日時や時刻を表します。この要素によりユーザエージェントは、記事の発行日時などを正確に判断することが可能となります。

属性

グローバル属性
詳細は グローバル属性 を参照してください。
datetime=datetime
LS/H5
この属性があればこの属性値が、無ければこの要素のコンテンツが日時を示します。
pubdate
[非推奨] H5d
この要素で示された日時が、この要素の直近の祖先の article要素、無ければドキュメント全体の発行日時とみなされます。HTML5 草案で検討されていましたが、最終勧告では削除されました。

使用例

<time>～</time> の間、または datetime属性に、コンピュータが解釈できる正式な日時を示す文字列を記述します。

HTML
<time>2011-06-11</time>
<time datetime="2011-06-11">2011年6月11日</time>

日時の箇所には下記の例のフォーマットで日時を記載します。日付と時刻の間には T または半角スペースを記述します。時刻の後ろにはタイムゾーンを記述します。Z は世界標準時を意味します。-09:00, -0900 は -9時間の差異を示します。

フォーマット
2011-12-31T23:59:59.999Z
2011-12-31T23:59:59.999-0900
2011-12-31T23:59:59.999-09:00
2011-12-31 23:59:59.999Z
2011-12-31 23:59:59.999-0900
2011-12-31 23:59:59.999-09:00

タイムゾーンは省略することができます。日付のみ、時刻のみを記述することもできます。

フォーマット
2011-12-31 23:59:59.999

時刻のみを記述することもできます。ミリ秒、秒を省略することができます。

フォーマット
23:59
23:59:59
23:59:59.999

日付のみを記述することもできます。年月のみ、月日のみを記述することもできます。

フォーマット
12-31
2011-12
2011-12-31
