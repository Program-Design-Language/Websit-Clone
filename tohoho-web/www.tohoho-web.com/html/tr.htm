<tr> - テーブルの行

トップ > HTMLリファレンス > <tr>

概要

形式
<tr>～</tr>
サポート
LS / H3 / e2 / Ch1 / Fx1 / Sa1 / Op1 / N2
カテゴリ
該当なし
親要素
thead, tbody, tfoot 要素
table 要素。ただし、caption, colgroup, thead 要素よりも後に記述すること。table 要素の子要素に tbody が存在しないこと。
子要素
th, td, スクリプトサポート要素
タグの省略
開始タグ: 必須
終了タグ: 他の tr 要素が直後に続く場合、もしくは、親要素に対する最後の要素の場合、省略可能。
属性
グローバル属性

説明

tr は Table Row の略です。テーブルの行を記述します。

属性

【一般属性】
グローバル属性
詳細は グローバル属性 を参照してください。
【ボーダー関連属性】
bordercolor=color
[非推奨] e2/Ch/Fx/Sa/Op
枠線の色を指定します。
bordercolordark=color
[非推奨] e2
立体的な枠線の暗い部分の色を指定します。
bordercolorlight=color
[非推奨] e2
立体的な枠線の明るい部分の色を指定します。
【背景関連属性】
background=url
[非推奨] Ch/Fx/Sa/Op/N4
背景画像を指定します。<table>, <th>, <td> でも指定可能です。
bgcolor=color
[非推奨] H4T-4T/e2/Ch/Fx/Sa/Op/N3
背景色を指定します。<table>, <th>, <td> でも指定可能です。
【配置関連属性】
align=align
[非推奨] H3-H4/e2/Ch/Fx/Sa/Op/N2
セル内データの横方向の配置を指定します。
center
中央表示(H3/e2/Ch/Fx/Sa/Op/N2)
char
指定文字揃え(H4)
justify
均等割付(H4)
left
左寄せ(H3/e2/Ch/Fx/Sa/Op/N2)
right
右寄せ(H3/e2/Ch/Fx/Sa/Op/N2)
valign=valign
[非推奨] H3-H4/e2/Ch/Fx/Sa/Op/N2
セル内データの縦方向の配置を指定します。
baseline
ベースライン揃え(H4/e2/Ch/Fx/Sa/Op/N2)
bottom
下揃え(H3/e2/Ch/Fx/Sa/Op/N2)
center
中央揃え(e2/Ch/Fx/Sa/Op/N2)
middle
中央揃え(H3/e2/Ch/Fx/Sa/Op/N2)
top
上揃え(H3/e2/Ch/Fx/Sa/Op/N2)
char=char
[非推奨] H4
位置揃えする文字を指定します。
charoff=n
[非推奨] H4
char 属性で指定した文字を、セルの左端からどのくらいの位置に表示するかのオフセットを指定します。
height=n
[非推奨] e4/Ch/Fx/Sa/Op
セルの高さをピクセルで指定します。
