Gimp とは？

Gimp（GNU Image Manipulation Program）は、Photoshop にも匹敵する機能を備えた無料の画像編集ソフトです。初心者でも、右のようなロゴをわずか数分で作成することができます。

数多くの優れたソフトウェアを開発してきた GNU プロジェクトによって UNIX 系 OS 用に開発されたのが始まりですが、フィンランドの tml 氏がこれを Windows に移植しました。元々日本語にも対応していましたが、いっちー氏が日本語専用インストールパッケージを作成されています。

これまではちょっと敷居の高かった Gimp ですが、Windows 版の登場により初心者でも簡単に使用できるようになってきました。このページでは、Windows 版 Gimp のほんの触りの部分を紹介します。

超簡単な使い方

[道具箱]
[The GIMP] というタイトルのウィンドウを [道具箱] と呼びます。[道具箱] は Gimp の一番中心となるウィンドウです。
[画像ウィンドウ]
[道具箱]→[ファイル]→[新規]→[了解] で [名称未設定] の [画像ウィンドウ] が作成されます。[画像ウィンドウ] はお絵描きのための画用紙のようなものです。
各種ダイアログを表示する
必要なダイアログが表示されていないときは、[道具箱]→[ファイル]→[ダイアログ] で各種ダイアログを表示してください。
ヘルプを見る
[道具箱]→[ヘルプ]→[ヘルプ]。ヘルプページ一番下の [Home] をクリックするとヘルプの全体像。ツールボタンやダイアログの説明を見たいときは、[道具箱]→[ヘルプ]→[状況対応ヘルプ] でマウスを？マークにして、調べたいボタンやダイアログの上でクリック。
筆で描く
[道具箱]→[絵筆で線を引く] ボタンをクリック。[ブラシ選択] ダイアログで好きなブラシを選び、[画像ウィンドウ] の上でマウスを押す→動かす→離す。
色を選択する
[道具箱]の左下にある[前景色]や[背景色]の領域をクリックすると[色選択]ダイアログが表示されるので、好きな色を選択。
消しごむで消す
[道具箱]→[消しごむで・・・] ボタンをクリック。[ブラシ選択] ダイアログで好きなブラシを選び、[画像ウィンドウ] の上でマウスを押す→動かす→離す。
画像を保存する
[画像ウィンドウ] の上でマウスの右クリック→[ファイル]→[保存]。Unisys 特許により GIF は扱えないので、JPEG や PNG などを選択。
ズームイン・ズームアウトする
[道具箱]→[ズームイン＆ズームアウト] ボタンをクリック。[画像ウィンドウ] の上をクリックでズームイン。Ctrl キーを押しながらクリックでズームアウト。
絵の場所を動かす
[道具箱]→[矩形領域を選択] ボタンをクリック。[画像ウィンドウ] の上で範囲を選択（マウスを押す→動かす→離す）。選択された範囲をマウスで好きな位置にドラッグ＆ドロップ（マウスで掴む→動かす→離す）。
選択を解除する
[矩形領域を選択]などで選択した領域は、選択されていない部分をマウスでクリックすることで解除できます。
文字を書く
[道具箱]→[画像に文字を追加] ボタンをクリック。[画像ウィンドウ] の上でクリック。プレビューに文字を入力。日本語の場合はフォントに [ＭＳ ゴシック] などを選択。[了解]。文字は選択された状態になるので、[The GIMP] で [矩形領域を選択] を選んで [画像ウィンドウ] の上でクリックすると、選択状態が解除される。[ツールオプション] ダイアログで [アンチエイリアシング] が効いていると、ギザギザのない綺麗なフォントになる。
かっこいいロゴを描く
[道具箱]→[拡張]→[Script-Fu]→[ロゴ]→[光沢] で [文字] に何かを入力。日本語ならフォントを [ＭＳ ゴシック] などに変更。[了解]。たったこれだけの操作で、このページの最初にあるような本格的なロゴを作成できちゃう。すごい。
用紙のサイズを変える
[画像ウィンドウ] を右クリック→[画像]→[キャンパスサイズ]。
縮小する
[画像ウィンドウ]→右クリック→[画像]→[拡大縮小]。
影をつける
[道具箱]→[矩形領域を選択する] ボタンで、影をつけたい部分をドラッグ＆ドロップで選択。[画像ウィンドウ]→右クリック→[Script-Fu]→[影]→[影付け]→[了解]。
