テキストや画像の表示座標を指定する

トップ > How To > テキストや画像の表示座標を指定する

説明

Internet Explorer 4.0やNetscape Communicator 4.0でサポートされたスタイルシートの中の、ポジショニングという技術を用いると、テキストや画像の表示位置をX, Yの座標で指定することができます。

実例

右上参照。

ソースコード

<span style="font-size:24pt; color:#ff9999;
     position:absolute; top:100px; left:350px;">ABC</span>
<span style="font-size:24pt; color:#99ff99;
     position:absolute; top:110px; left:360px;">ABC</span>
<span style="font-size:24pt; color:#9999ff;
     position:absolute; top:120px; left:370px;">ABC</span>

動作確認

IE4.0/5.0(Win)、IE4.0(Mac)、Netscape 4.0(Win)での動作を確認しています。JustView 3.01 などでは動作しません。
