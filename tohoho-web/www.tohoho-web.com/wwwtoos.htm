Windows 3.1/95/98/Me/NT/2000/XP

★★★ [Win] 製品
http://www.microsoft.com/Japan/

Microsoft社のOSです。あまり多くを語る必要がないくらい、広く利用されています。（2001.10.13更新）

Macintosh・Mac・マック

★★ [Mac] 製品
http://www.apple.co.jp/

Macintosh は OS の名前ではないですね。MacOS 9 までは Apple 社の独自 OS でしたが、Mac OS X（テン）では、FreeBSD をベースに大きく変更されているようです。Windows に比べるとシェアもアプリケーションも少ないですが、デザイン系の方を中心に根強い人気があります。（2002.10.13更新）

UNIX

★ [UNIX] 製品/FREE

Webサーバによく利用されているOSです。一口に UNIX と言っても、Soralis（Sun社）、HP-UX（HP社）、AIX（IBM社）、フリーの Linux や、FreeBSD など、様々な UNIX系 OS が混在しています。（→ 「とほほのUNIX入門」）（2000.12.17更新）

Linux

★ [UNIX] FREE
http://www.linux.or.jp/

パソコンで動くフリーのUNIX系OSです。Linux は OS の中核部分のみを言いますが、これにいろいろな無料アプリケーションを追加して使いやすくしたものが、RedHat、TurboLinux などのディストリビューションとして配布されています。雑誌の付録 CD-ROM などにも添付されて無料で使用できるので、ハードディスクに余裕があれば、ちょっと遊んでみるのもよいのではないでしょうか。（2000.12.17更新）

FreeBSD

★ [UNIX] FREE
http://www.jp.freebsd.org/

Linux 同様、パソコンで動くフリーのUNIXです。シェアは Linux の方が多いようです。（2000.12.17更新）
