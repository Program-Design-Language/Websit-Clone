Webコンポーネント(Web Components)

トップ > アラカルト > Webコンポーネント(Web Components)

概要

Webコンポーネント(Web Components) は、カプセル化されたコンポーネントを定義し、これを再利用するための技術の総称です。コンポーネントは、HTML、CSS、JavaScript などから構成されるコンテンツ部品です。

下記などの技術から構成されます。

  • カスタム要素(Custom Elements)
  • シャドーDOM(Shadow DOM)
  • HTMLインポート(HTML imports)
  • HTMLテンプレート(HTML Template)
