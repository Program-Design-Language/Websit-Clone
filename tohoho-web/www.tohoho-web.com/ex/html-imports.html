HTMLインポート(HTML imports)

トップ > アラカルト > HTMLインポート(HTML imports)

概要

HTMLインポート(HTML imports) は、Webコンポーネント(Web Components) と呼ばれる技術群のひとつです。他のファイルに記載された HTML コンテンツを読み込みます。Chrome 36、Opera 23 などでサポートされていましたが、互換性の維持等に問題があったのか、現在ではサポートが中止される方向にあります。

サンプル

HTML
<head>
  <link rel="import" href="example.html">
</head>
