FFFTP Version 1.34

★★★ [Win] フリーソフト 日本語
http://www2.biglobe.ne.jp/~sota/
フリーソフト。完全日本語なので初心者にも簡単。エクスプローラからのドラッグ＆ドロップ、転送時の漢字コード変換、サブディレクトリ一括転送、一度中断したダウンロードを途中から再開、更新されたファイルのみを転送、転送するファイル名をワイルドカードや正規表現で指定、転送中のファイルの追加削除・・・・などなど、機能面も文句無しの ★★★。

NextFTP Ver1.97

★★ [Win] シェアウェア(2,000円) 日本語
http://www.toxsoft.com/
日本語ヘルプも充実、更新ファイルだけをアップロード、フォルダ自動作成、転送時に漢字コード変換など、高機能。

WS_FTP Ver6.0J

★ [Win] 製品版/フリーソフト
http://www.ipswitch.com/
http://wwwcsoft.kgt.co.jp/product/wsftp/index.html
製品版(6,800円～)のWS_FTP Proと、無料のWS_FTP LE(Light Edition)がある。後者は英語版しか用意されておらず、若干の機能制限があるが、教育関係者と非営利目的のホームユースの場合に限り無料で使用可能。(2000.11.2)

FTP Explorer

★ [Win] シェアウェア
http://www.ftpx.com/
シェアウェアだが教育関係、ホームユースであれば無料。Windows 95のエクスプローラーに似た操作感。基本的には英語版だが、日本語版用のリソースもある。

CuteFTP 4.0

★ [Win] シェアウェア($39.95)
http://www.cuteftp.com/
エクスプローラ風。(2000.11.2)

WinFTP

★ [Win] フリーソフト
英語版のみ。作者も最近はほとんどメンテナンスしていないらしい。

FTP.EXE

★ [Win]
Windows 95に標準で添付されている。MS-DOSプロンプト上で使用する。cd、put、get などのコマンドを覚える必要があるので、上級キーボード派好み。

Fetch 3.0.3J2

★★★ [Mac]
http://www.hart.co.jp/fetch/
Macでは定番の逸品。オリジナルの英語版は$25(学術関係は無料)のシェアウェア、日本語版は2,500円のシェアウェア。製品版5,000円もある。Mac版FTPではトップシェアを保っているようだ。(1999.6.27)

Interarchy 3.8

★ [Mac]
http://www.interarchy.com/
昔は Anarchie という名前だったが、Interarchy に改名。シェアウェア($25)。日本語用パッチもあるらしい。(2000.11.2)
