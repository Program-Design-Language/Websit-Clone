クライアントプル

Netscape 2.0やIE3.0以降では、クライアントプル(Client Pull)という技術を用いて、ページを定期的に自動再表示することができます。

方法は簡単で、HTML文書のヘッダ部分に次のような<meta>タグを記述するだけで、5秒毎に自動再表示が行われます。

<html>
<head>
<title>クライアントプルのテスト</title>
<meta http-equiv=Refresh content=5>
</head>
<body>
現在の時刻は
<script>
d = new Date();
document.write(d.toLocaleString());
</script>
です。
</body>
</html>
