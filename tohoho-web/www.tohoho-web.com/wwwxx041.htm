セレクトメニューからジャンプする

トップ > How To > セレクトメニューからジャンプする

説明

JavaScript を用いた基本的なテクニックのひとつです。セレクトボックスで指定したページにジャンプします。Netscape 4.0以降、IE4.0 以降で動作します。

実例

ソース

<select onchange="location.href = this.options[selectedIndex].value">
 <option value="">ページにジャンプ
 <option value="http://www.yahoo.co.jp/">Yahoo
 <option value="http://www.google.co.jp/">Google
 <option value="http://www.goo.ne.jp/">goo
</select>
