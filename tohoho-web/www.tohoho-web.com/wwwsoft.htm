とほほのCGIソフト集

このサイトで配布しているCGI関連フリーソフト（私用・商用を問わず、利用・改造・流用・再配布可能）です。ただし、jcode.pl（Utashiro氏開発） と mimew.pl（生田氏開発） が添付されている場合は、それぞれの使用条件に従ってください。

  • jcode.pl - 大きな改変が無ければ利用と再配布が可能。
  • mimew.pl - 改変したものを再配布する場合は注意事項あり。

うまくダウンロードできない場合は、マウスの右クリック（Mac の場合はしばらくクリック）メニューから [対象をファイルに保存] などのメニューからダウンロードしてください。

サーバによっては jcode.pl のバージョンが古いと動かないという連絡をいただきました。うまく動かない場合は下記から新しい jcode.pl を入手して置き換えてみてください。

  • https://ja.osdn.net/projects/beetas/releases/628
