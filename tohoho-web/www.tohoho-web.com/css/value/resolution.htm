CSS - <resolution>

トップ > CSSリファレンス > <resolution>

説明

レゾリューションを表す単位として、下記が定義されています。

単位 説明
dpi 1インチあたりのドット数。
dpcm 1cm あたりのドット数。
dppx 1px あたりのドット数。

使用例

CSS
@media (min-resolution: 2dppx) {
  ...
}
