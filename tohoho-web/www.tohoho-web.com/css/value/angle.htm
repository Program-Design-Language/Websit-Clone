CSS - <angle>

トップ > CSSリファレンス > <angle>

説明

角度を指定します。下記の単位が定義されています。

単位 説明
deg 角度を度で表します。全円は 360deg となります。
grad 角度をグラジアンで表します。全円は 400grad となります。
rad 角度をラジアンで表します。全円は 2πラジアン(約6.2832rad) となります。
turn 角度を回転数で表します。全円は 1turn となります。

0deg, 0grad, 0rad, 0tum は上方をさし、時計回りに増加します。<length> とは異なり、0deg, 0grad, 0rad, 0tum の場合であっても単位を省略することはできません。

使用例

HTML
<div style="transform:rotate(15deg); width:60px;">こんにちわ</div>
