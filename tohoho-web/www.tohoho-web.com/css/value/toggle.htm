CSS - toggle()

トップ > CSSリファレンス > toggle()

概要

形式toggle(value, ...)
サポート[非推奨]

説明

ネストの深さに応じた値を指定します。CSS3 の草案で定義されましたが、実装例が少ないことから廃案となりました。

使用例

下記の例では、リストスタイルとして、一番外側の ul では disc、その内側の ul では circle、その内側の ul では square、その内側の ul では最初に戻って disc を指定します。

CSS
ul {
  list-style-type: toggle(disc, circle, square);
}
