CSS - <frequency>

トップ > CSSリファレンス > <frequency>

説明

周波数を表す単位として、下記が定義されています。

単位 説明
Hz ヘルツ。
kHz キロヘルツ。1kHz = 1000Hz。

使用例

CSS
* {
  voice-pitch: 250Hz;
}
