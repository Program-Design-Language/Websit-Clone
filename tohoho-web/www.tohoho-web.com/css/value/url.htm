CSS - <url>

トップ > CSSリファレンス > <url>

説明

外部ファイルのURLを指定します。

body { background-image: url(../image/back.gif); }
body { background-image: url(http://www.example.com/image/back.gif); }
url(url)
URL を指定します。

使用例

CSS
.d1 {
  color: black;
  background-image: url(../image/back.gif);
  width: 100px;
  height: 100px;
  border: 1px solid gray;
}
HTML
<div class="d1">こんにちわ</div>
表示
こんにちわ
