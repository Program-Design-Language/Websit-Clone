CSS - initial

トップ > CSSリファレンス > initial

説明

CSS3 では、すべての属性で initial という値を指定することができます。initial は、各属性の初期値を意味します。属性 all に initial を指定することで、すべての属性指定をリセットすることができます。

サポート (説明)

CSSIE/EdgeFirefoxChromeOperaSafari
CSS312191.0153.2

使用例

下記の例では、親要素(classA)で指定されたテキストの色を、子要素(classB)ではそのまま継承し、子要素(classC)では初期値(通常は黒)に戻しています。

CSS
.classA {
  color: red;
}
.classB {
  margin-left: 10px;
}
.classC {
  color: initial;
  margin-left: 10px;
}
HTML
<div class="classA">
 この部分は赤字になります。
 <div class="classB">
  通常は子要素も赤字になります。
 </div>
 <div class="classC">
  color: initial を指定すると初期値(通常は黒)に戻ります。
 </div>
</div>
表示
この部分は赤字になります。
通常は子要素も赤字になります。
color: initial を指定すると初期値(通常は黒)に戻ります。
