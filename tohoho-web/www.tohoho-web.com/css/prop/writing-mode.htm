CSS - writing-mode

トップ > CSSリファレンス > writing-mode

概要

属性名 writing-mode
値 lr-tb | tb-rl | horizontal-tb | vertical-rl | vertical-lr
初期値horizontal-tb
適用可能要素すべての要素
継承継承する
メディアVisual / Writing Mode
サポートC3 / e5 / Ch / Fx / Ed / Op / Sa

説明

縦書きを指定します。Internet Explorer 5 で実装されたものを CSS3 でも採用が検討されていますが、値の名称が変更される可能性があります。

値説明
lr-tb横書き(左→右)で上→下に改行します。left to right, top to bottom。(e5/Ch/Fx/Ed/Op/Sa)
tb-rl縦書き(上→下)で右→左に改行します。top to bottom, right to left。(e5/Ch/Fx/Ed/Op/Sa)
horizontal-tb横書きで上→下に改行します。(C3/Ch/Fx/Ed/Op/Sa)
vertical-rl縦書きで右→左に改行します。(C3/Ch/Fx/Ed/Op/Sa)
vertical-lr縦書きで左→右に改行します。(C3/Ch/Fx/Ed/Op/Sa)

使用例

CSS
.sample {
    margin: 1em;
    padding: 5px;
    height: 100px;
    width: 200px;
    border: 1px solid #666666;
}
.sample.lr-tb { writing-mode: lr-tb; }
.sample.tb-rl { writing-mode: tb-rl; }
.sample.horizontal-tb { writing-mode: horizontal-tb; }
.sample.vertical-rl { writing-mode: vertical-rl; }
.sample.vertical-lr { writing-mode: vertical-lr; }
