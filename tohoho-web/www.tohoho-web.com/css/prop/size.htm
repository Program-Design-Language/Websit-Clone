CSS - size

トップ > CSSリファレンス > size

概要

属性名 size
値 auto | [ <width> <height>? ] | [ <page-size> || [ portrait | landscape ] ]
値の詳細 <page-size> = A5 | A4 | A3 | B5 | B4 | letter | legal | ledger
初期値 auto
適用可能要素スクロールバーを持つ要素
継承継承しない
メディアVisual / Page
サポートC3

説明

印刷時のページのサイズを指定します。@page { ... } の中で使用します。CSS2.0 で定義されましたが、CSS2.1 で破棄され、CSS3 で再定義される予定です。

値説明
auto横幅・縦幅を自動計算します。
<width>横幅を 21.0cm などの長さの単位で指定します。
<height>縦幅を 29.7cm などの長さの単位で指定します。省略時は横幅と同じになります。
A5A5 サイズで印刷します。
A4A4 サイズで印刷します。
A4A3 サイズで印刷します。
B5B5 サイズで印刷します。
B4B4 サイズで印刷します。
letter米国等でよく使用されるレターサイズ(8.5×11in)で印刷します。
legal米国等でよく使用されるリーガルサイズ(8.5×14in)で印刷します。
ledger米国等でよく使用されるレジャーサイズ(17×11in)で印刷します。
portrait縦長の方向で印刷します。
landscape横長の方向で印刷します。

使用例

CSS
@page {
    size: A4 landscape;
}
