CSS - column-gap

トップ > CSSリファレンス > column-gap

概要

属性名 column-gap
値 <length> | normal
初期値normal
適用可能要素マルチカラムエレメント
継承継承しない
メディアvisual

サポート (説明)

CSSIE/EdgeFirefoxChromeOperaSafari
CSS31024153.1

説明

マルチカラムのカラム間の隙間の幅を指定します。カラム間に罫線がある場合は、隙間の中央に描画されます。

値説明
<length>マルチカラムのカラム間の幅を指定します。
normalブラウザによる標準の隙間を開けます。推奨値は 1em です。

使用例

CSS
.multicol {
  column-count: 3;
  column-gap: 5em;
}
HTML
<div class="multicol">
いろはにほへとちりぬるを...
</div>
