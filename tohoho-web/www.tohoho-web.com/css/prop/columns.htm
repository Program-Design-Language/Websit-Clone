CSS - columns

トップ > CSSリファレンス > columns

概要

属性名 columns
値 <width> || <count>
初期値個々の属性を参照
適用可能要素table を除く非置換のブロックレベル要素、テーブルセル、インラインブロック
継承継承しない
メディアvisual

サポート (説明)

説明

マルチカラムの幅、カラム数をまとめて指定します。

値説明
<width>カラムの幅を指定します。詳細は column-width を参照してください。
<count>カラムの個数を指定します。詳細は column-count を参照してください。

使用例

CSS
.multicol {
  columns: 3;
}
HTML
<div class="multicol">
いろはにほへとちりぬるを...
</div>
