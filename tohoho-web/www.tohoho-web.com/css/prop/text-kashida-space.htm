CSS - text-kashida-space

トップ > CSSリファレンス > text-kashida-space

概要

属性名 text-kashida-space
値 <percentage>
初期値 0%
適用可能要素すべての要素
継承継承する
メディアVisual / Text
サポートe5.5

説明

アラビア語などのテキスト配置を最善化する際に用います。test-align: justify および text-justify: kashida と共に指定する必要があります。

値説明
<percentage>テキストの余白の比率を 30% のようなパーセントで指定します。

使用例

CSS
.sample-kashida {
    text-align: justify;
    text-justify: kashida;
    text-kashida-space: 0%;
}
