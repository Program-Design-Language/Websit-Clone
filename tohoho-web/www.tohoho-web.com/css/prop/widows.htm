CSS - widows

トップ > CSSリファレンス > widows

概要

属性名 widows
値 <integer>
初期値2
適用可能要素ブロックレベル要素
継承継承する
メディアVisual / Page
サポートC2 / e8 / Op9.2

説明

印刷時、ページの上部の段落に最低限印字すべき行数を指定します。ページ下部の段落の最低限印字行数を指定する orpansも参照してください。

値説明
<integer>行数を 4 などの整数値で指定します。

例えば、下記のような印刷となる場合、Page.2 上部の段落３が 3行しか印字できないため、widows: 4 を指定した場合、前のページの最後の 1行がこのページに印刷されます。
