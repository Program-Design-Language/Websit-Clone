CSS - font-synthesis

トップ > CSSリファレンス > font-synthesis

概要

属性名 font-synthesis
値 none | [ weight || style ]
初期値weight style
適用可能要素すべての要素
継承継承する
メディアVisual
サポートC3 / Fx34

説明

使用するフォントが太字や斜体をサポートしていない場合に、無理やり太字や斜体表示するか否かを指定します。

値説明
none太字や斜体文字を自動生成しません。
weight使用するフォントが、font-weight で指定された太さの太字をサポートしていない場合、通常のフォントから太字を自動生成して表示します。
style使用するフォントが、font-style で指定された斜体などのスタイルをサポートしていない場合、通常のフォントから指定されたスタイルの文字を自動生成して表示します。

使用例

CSS
.bold-sample {
  font-weight: bold;
  font-synthesis: none;
}

関連項目

font-weight / font-style
