CSS - column-rule

トップ > CSSリファレンス > column-rule

概要

属性名 column-rule
値 <width> || <style> || [ <color> || transparent ]
初期値個々の属性を参照
適用可能要素マルチカラムエレメント
継承継承しない
メディアvisual

説明

マルチカラムのカラム間の罫線の幅、スタイル、色をまとめて指定します。

値説明
<width>罫線の幅を指定します。詳細は column-rule-width を参照してください。
<style>罫線のスタイルを指定します。詳細は column-rule-style を参照してください。
<color>罫線の色を指定します。詳細は column-rule-color を参照してください。
transparent透明色を指定します。

使用例

CSS
.multicol {
  columns: 3;
  column-rule: 1px dashed #666;
}
HTML
<div class="multicol">
いろはにほへとちりぬるを...
</div>
