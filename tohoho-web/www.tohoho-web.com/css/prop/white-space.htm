CSS - white-space

トップ > CSSリファレンス > white-space

概要

属性名 white-space
値 normal | pre | nowrap | pre-wrap | pre-line
初期値各要素によって異なる
適用可能要素すべての要素
継承継承する
メディアVisual / Text
サポートC1 / e6 / N4 / Fx1 / Ch1 / Op4 / Sa1

説明

要素の中のテキストを、自動改行するか、複数の空白をひとつの空白に置換するか、改行を空白に置換するかを指定します。

値説明
normal自動改行します。複数の空白・タブ・改行をひとつの空白に置換します。(C1/e6/N4/Fx1/Ch1/Op4/Sa1)
pre自動改行しません。空白・タブ・改行はそのまま表示します。(C1/e6/N4/Fx1/Ch1/Op4/Sa1)
nowrap自動改行しません。複数の空白・タブ・改行をひとつの空白に置換します。(C1/e6/N4/Fx1/Ch1/Op4/Sa1)
pre-wrap自動改行します。空白・タブ・改行はそのまま表示します。(C2/e/Fx/Ch/Op/Sa)
pre-line自動改行します。複数の空白・タブをひとつの空白に置換します。ただし改行はそのまま表示します。(C2/e/Fx/Ch/Op/Sa)

使用例

CSS
.sample-normal {
    white-space: normal;
    margin: 1em;
    width: 300px;
    border: 1px solid #666666;
}
.sample-pre {
    white-space: pre;
    margin: 1em;
    width: 300px;
    border: 1px solid #666666;
}
.sample-nowrap {
    white-space: nowrap;
    margin: 1em;
    width: 300px;
    border: 1px solid #666666;
}
.sample-pre-wrap {
    white-space: pre-wrap;
    margin: 1em;
    width: 300px;
    border: 1px solid #666666;
}
.sample-pre-line {
    white-space: pre-line;
    margin: 1em;
    width: 300px;
    border: 1px solid #666666;
}
