CSS - marker-offset

トップ > CSSリファレンス > marker-offset

概要

属性名 marker-offset
値 <length> | auto
初期値auto
適用可能要素すべての要素
継承継承しない
メディアVisual / Box
サポートC2.0

説明

リスト項目の先頭に表示するマーカーの位置を指定します。CSS2.0 で定義されましたが、CSS2.1 では削除されました。CSS3 では、marker-offset の代わりに ::marker 疑似要素がサポートされる予定です。メジャーなブラウザでこの属性をサポートしているものはありません。

値説明
<length>マーカーのオフセット位置を 1em などの長さの単位で指定します。
autoオフセット位置を自動計算します。

使用例

CSS
li:before {
    display: marker;
    marker-offset: 3em;
    content: counter(mycounter, lower-roman) ".";
    counter-increment: mycounter;
}
