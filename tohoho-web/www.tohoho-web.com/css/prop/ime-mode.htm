CSS - ime-mode

トップ > CSSリファレンス > ime-mode

概要

属性名 ime-mode
値 auto | active | inactive | disabled
初期値auto
適用可能要素入力可能要素
継承継承しない
メディアVisual / UI
サポートe5 / Fx3

説明

input要素の入力領域などにフォーカスが移動したとき、日本語変換機能(IME:Input Method Editor)を有効にするか否かを指定します。

値説明
auto自動。
activeIMEをオンにします。
inactiveIMEをオフにします。
disabledIMEを使用禁止にします。

使用例

HTML
<div><input type="text" value="auto" style="ime-mode:auto"></div>
<div><input type="text" value="active" style="ime-mode:active"></div>
<div><input type="text" value="inactive" style="ime-mode:inactive"></div>
<div><input type="text" value="disabled" style="ime-mode:disabled"></div>
