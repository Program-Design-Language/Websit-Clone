CSS - list-style-image

トップ > CSSリファレンス > list-style-image

概要

属性名 list-style-image
値 <url> | none
初期値none
適用可能要素li などのリストアイテム要素
継承継承する
メディアVisual / List
サポートC1 / e4 / N4 / Fx1 / Ch1 / Op3.5 / Sa1

説明

リストのマーカーに使用する画像ファイルを指定します。

値説明
<url>マーカーの画像を指定します。
noneマーカーを表示しません。

使用例

HTML
<ul style="list-style-image:url(../image/mark.gif)">
  <li>HTML
  <li>CSS
  <li>JavaScript
</ul>
