CSS - isolation

トップ > CSSリファレンス > isolation

概要

属性名 isolation
値 auto | isolate
初期値auto
適用可能要素全ての要素 (SVGの場合はコンテナ要素、グラフィック要素、グラフィック参照要素)
継承継承しない
メディアVisual
サポートC2 / Fx36 / Ch41

説明

SVG においては、要素が独立(isolated)するか否かを指定します。CSS においては、要素が新しいスタックコンテキストを作成するか否かを指定します。

値説明
autoスタックコンテキストを作成しません。
isolate新しいスタックコンテキストを作成します。

スタックコンテキストに関する詳細は下記を参照ください。

  • https://developer.mozilla.org/ja/docs/Web/Guide/CSS/Understanding_z_index/The_stacking_context
