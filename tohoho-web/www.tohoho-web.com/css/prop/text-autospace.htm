CSS - text-autospace

トップ > CSSリファレンス > text-autospace

概要

属性名 text-autospace
値 none || ideograph-alpha || ideograph-numeric || ideograph-parenthesis || ideograph-space
初期値 none
適用可能要素すべての要素
継承継承しない
メディアVisual / Text
サポートe5.5

説明

漢字のような表意文字と半角英数字のような非表意文字の間にほんの少しだけ隙間を入れるかどうかを指定します。

値説明
none隙間を空けません。
ideograph-alpha表意文字とアルファベットの間に隙間を空けます。
ideograph-numeric表意文字と数字の間に隙間を空けます。
ideograph-parenthesis表意文字と括弧の間に隙間を空けます。
ideograph-space表意文字と表意文字の間の隙間を広くします。

使用例

CSS
.sample-none {
    text-autospace: none;
}
.sample-alpha {
    text-autospace: ideograph-alpha;
}
.sample-numeric {
    text-autospace: ideograph-numeric;
}
.sample-parenthesis {
    text-autospace: ideograph-parenthesis;
}
.sample-space {
    text-autospace: ideograph-space;
}
.sample-all {
    text-autospace: ideograph-space ideograph-numeric ideograph-parenthesis ideograph-space;
}
