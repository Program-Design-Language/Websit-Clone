CSS - all

トップ > CSSリファレンス > all

概要

属性名 all
値 initial | inherit | unset | revert
初期値個々の属性を参照
適用可能要素個々の属性を参照
継承個々の属性を参照
メディア個々の属性を参照
アニメーション個々の属性を参照

サポート (説明)

CSSIE/EdgeFirefoxChromeOperaSafari
基本CSS3(未)273724(未)

説明

unicode-bidi と direction を除くすべてのプロパティの値をまとめて設定します。ページ内に他のデザインのコンテンツを取り込む際、既存のスタイルをリセットする際などに有効です。値には、すべての属性で共通な、下記の値を指定することができます。

値説明
initial属性値を初期値に戻します。
inherit属性値を親要素から継承します。
unset属性値を初期値に戻します。ただし、ただし、親要素から継承される値は継承された値となります。
revert属性値をオリジンの規定値に戻します。ユーザエージェントオリジンの場合は unset と同じ意味になります。ユーザオリジンの場合はユーザが指定した規定値、オーサーオリジンの場合はオーサーが指定した規定値となります。CSS Cascading and Inheritance Level 4 でサポートされた値で、まだあまりサポートされていません。

使用例

CSS
.parent {
  color: red;
  font-weight: bold;
}
.child1 {
  border:1px solid #999;
  margin-top: 10px;
  margin-bottom: 10px;
}
.child2 {
  all: initial;
  border:1px solid #999;
}
HTML
<div class="parent">
  この部分には、赤い太字のスタイルが設定されています。
  <div class="child1">
    通常は、子要素にもスタイルが継承されます。
  </div>
  <div class="child2">
    all:initial を行うと、親要素から継承したスタイルがすべてリセットされます。
  </div>
</div>
表示
この部分には、赤い太字のスタイルが設定されています。
通常は、子要素にもスタイルが継承されます。
all:initial を行うと、親要素から継承したスタイルがすべてリセットされます。
