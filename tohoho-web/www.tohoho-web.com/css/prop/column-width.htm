CSS - column-width

トップ > CSSリファレンス > column-width

概要

属性名 column-width
値 <length> | auto
初期値auto
適用可能要素table を除く非置換のブロックレベル要素、テーブルセル、インラインブロック
継承継承しない
メディアvisual

サポート (説明)

CSSIE/EdgeFirefoxChromeOperaSafari
CSS3101.5(-moz)
52		4(-webkit)11.13(-webkit)

説明

マルチカラムの各カラムの幅を指定します。

値説明
<length>最適なカラム幅を指定します。実際のカラム幅は指定したものよりも広くなったり狭くなったりします。
autoカラムの幅を他の属性から自動的に算出します。

使用例

CSS
.multicol {
  column-count: 3;
  column-width: 8em;
}
HTML
<div class="multicol">
いろはにほへとちりぬるを...
</div>
