CSS - @important

トップ > CSSリファレンス > @important

概要

宣言名 @important
構文
... { ... !important }
サポート C1 / e4 / N6 / Fx1 / Ch1 / Op3.5 / Sa1

説明

!important は、スタイルが適用される優先順位を最優先にする際に用いられます。下記の例では、通常は div { ... } よりも div.classA { ... } の方が優先度が高いため、color: blue が適用されますが、!important を宣言することにより、div.classA { ... } よりも div { ... } の方が優先度が高くなります。

CSS
div.classB { color: blue; }
div { color: red !important; }

スタイルシートの優先順位に関しては、優先度の順位 も参考にしてください。
