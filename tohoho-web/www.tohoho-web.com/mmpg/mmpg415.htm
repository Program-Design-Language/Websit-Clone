おススメポイント（平成１０年３月１９日）

一般利用で行くなら、メインは３・４・５階。保護者付き添いで子供たちが自由に遊ぶ、というところが「近所の児童館」の大きいの、という感じ。
３階はプレイホール。アスレチック「わくわくらんど」で遊んだり「コンピュータプレイルーム」でソフトを借りて遊ぶこともできるよ。
４階は「音楽ロビー」で太鼓をたたいたり、ボランティアのスタッフと遊んだりできるよ。「ＡＶライブラリー」で好きなビデオを選んでブースで見ることもできるよ。
５階屋上を利用した遊び場。三輪車などの乗り物が乗れます。「プレイポート」はボールのプールや滑り台があるようですが、この日は工事中で使用できませんでした。
曜日によって、お話読み聞かせや、イベント、講座もあるようです。地下にはプールや体育施設、また、幼児保育に関する大人向けの講座などもあるようです。
