おススメポイント（平成８年７月２８日）

館の前では機関車が展示されています。ここの 山の手線・新幹線の運転シミュレーションは 子供の列ができる人気コーナー。鉄道以外にも船舶・自動車・航空機など展示されています。男の子が絶対喜ぶ博物館ですね。 パパも夢中になりますよ。
