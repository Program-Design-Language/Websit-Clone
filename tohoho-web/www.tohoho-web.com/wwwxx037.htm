説明

背景画像をウィンドウの大きさにあわせて、ウィンドウいっぱいの大きさに引き伸ばして表示します。

ソースコード

<html>
<head>
<title>TEST</title>
</head>
<style>
body {
  background-image: url(image/back.gif);
  background-size: cover;
}
</style>
<body>
 ：
本文
 ：
</body>
</html>
